国企的作用是产生对经济增长的缓冲

[数学] 2001-06-12

我喜欢拿人体这种系统和社会这种系统进行比较。那么，大自然创造的人体 这种奇妙系统有一个特点，体重增加到一定程度就不增加了。比方说，一个人从 0 岁到20岁之间身体增加了五十公斤，那么按比例计算，到八十岁的时候一个人 就应当有二百公斤。但实际上并不这样，这就说明人体的自我调节功能，有意地 控制了细胞的分裂和繁殖，不让它们增长过快。任何生物如果让所有细胞都以最 大的速度分裂繁殖，其结果将是灾难性的。

而我认为，国企的作用，应当是有意识地在放缓经济增长中起作用，不让经 济增长过快，因此，国企的目标就不应当是盈利，而应当是保本，简单再生产就 可以。

社会主义发展到一定阶段，就应当进入简单再生产阶段，人口不再增加，生 产总量不再增加，人均产值也不再增加。

那么，这不是阻碍了生产力发展么？错，生产力还是继续发展，经济增长和 生产力发展是两回事，完全可以控制做到生产力还是不断发展的，而经济却是不 增长的，而人民生活水平却仍然是不断提高的。

在简单再生产的情况下，人民生活水平的提高表现在几个方面。

一个方面是闲暇时间的增加。看一个人生活的好不好，并不是只看他住的怎 样，穿得怎样，吃得怎样，还包括看他劳动时间是否缩短，自由的时间是否增加。

劳动的本质，就是用不自由换取自由，因此，用越来越少的不自由换取越来 越多的自由，这社会就是进步的。

因此，考察一个社会的两个时期，如果发现这两个时期的人民吃的穿的住的 基本上一样，没有变化，是否说明这个社会在停滞呢？也不见得，还要比较这两 个时期的闲暇情况。如果一个时期人民被逼要每天不停干活十几个小时，还经常 挨打受骂，而另一个时期人民只需干很少时间的活儿，没有压迫没有剥削，那么 另一个时期的人民显然要幸福得多。

因此一些国际上所谓公认的贫富悬殊的所谓“基尼系数”其实并不科学，在 比较贫富时没有将闲暇情况加入在内考虑。

一种说法是缺少了对利润的追求，人们就没有发展生产力的积极性。错。就 说苏联吧，人类第一颗人造卫星不还是在苏联上的天么？能够说苏联的科学技术 没有一直在增长么？只不过这些增长没有体现在经济的增长上而已。再拿美国作 例子，美国的警察这几十年来装备也一直在改善，破案技术也一直在改善，而这 种改善是因为有利润的刺激么？其实也不是，警察部门也不是盈利单位，改善了 设备警察也不会加薪。这就已经说明缺少对利润的追求仍然可以有对生产力发展 的追求。中国这方面的例子就更多。

有人说国企占有了大量的资源，却没有大量的产出，效益很低，这要看怎么 计算。国企是社会主义革命的产物，即工人直接掌握生产资料，生产的目的是为 了改善工人自己的生活。那么和私企相比。私企计算利润的方法，是将总产值减 去成本，成本越低利润越高。而私企是将员工工资当作成本的，那么员工工资越 高成本越高，利润就越低，那么作为老板当然希望员工的工资能低就低。而国企 则挣了钱是大家的，大家分，工资就会上去，当然利润就没有办法和私企相比， 可是不等于生产力没有增加。

实际上国企应当是零利润才对，工人创造了财富，工人享受全部财富，没有 利润就是没有剥削，简单再生产，不就是这样么？

国企在另一个方面对工人工作的条件改善，就是工作时间不象私企那样长， 这本身也是对社会的贡献。因此国企本来就没有必要同私企比利润，也不可能比 得过。

再回顾1998年的亚洲金融危机，亚洲各国纷纷遭难，可是中国的经济屹立不 倒，以至于国内许多人纷纷到东南亚各国旅游惊呼人民币值钱，很上算，正是在 这几年国内私人出国旅游不断增加。我听到不止一个人说中国之所以躲过大难， 是由于因祸得福，是因为中国的经济不规范，不那么象资本主义。那么又谈何祸 呢？想象一下，如果中国早在1990年就干脆彻底地向资本主义转，所有的国企全 部私有化，到了1998年，全部私有化的中国能不能顶得住亚洲金融风暴？我认为 不能。因此，中国之所以能够顶得住亚洲金融风暴，国企在其中是起了作用的。

一辆没有制动刹车装置的汽车或者自行车，有没有人敢用它？我看没有人敢 用。一个好的汽车或者自行车，必然有好的刹车装置。那么，研究刹车装置，就 会发现刹车是在浪费能源，本来产生了向前的动力，为什么要通过刹车来将其白 白浪费？可是事情的本来原理就是这个样子。比如原子能的利用，也是要通过石 墨来进行制动，不让核裂变那么快，人类才有可能和平利用原子能。

普遍的看法总是以为改革总是好的，保守总是不好的，其实，保守也起到了 一种刹车或者制动的作用，中国的改革之所以能够成功，保守派功不可没。

而在经济上，国企就起到了对经济制动的作用。美国对经济的制动手段就是 加息减息，而中国使用国企进行制动，就会使经济更加稳定，抵御经济危机的能 力就很强。国企的存在也使私企对工人的剥削不那么过于残酷，因为国企将工人 的水平强行地稳定到某一个水平线以上。如果私企要从国企挖人，只能出更高的 薪水。如果没有国企的存在，私企的工人状况有可能迅速下降。

按照马克思的经济学，价值其实就是社会必要劳动时间。那么一个正常的社 会主义国家总产值就是应当不增加的，除非人口增加。而技术的改进使人们生产 的时间减少，则总产值也就减少。因此，社会主义的经济还应当负增长。当然， 生产力总是增加的，人类对自然界的认识，科学的发展，新型产品的出现还是不 断会有的。
