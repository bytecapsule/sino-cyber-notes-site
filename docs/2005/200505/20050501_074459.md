就绝对优势兵力问题再说一些

[数学] 2001-05-07

我发现有一些网友没有理解我的想法，或者说，当我提出一个简单的模型之 后，他们以为我只是按这个简单的模型来行事了。其实，复杂的模型也是可以计 算的，但我不希望强国论坛的文章出现一大堆数学公式。

我在上文中只打了步兵相互射击的比方。那么这一次现实一点，打一个轰炸 机与防空导弹的博奕情况。也还是将模型简化一些，但是，如果复杂一些，也完 全可以通过计算机进行仿真计算的。

假设台湾方有一个防空导弹阵地，其中有10个发射架。当然，我这么说是为 了通俗，如果吓人一些，也可以说有N 个发射架。假设这10个发射架的每个平均 每分钟能够发射一枚导弹，当然在正式的论文里是发射K 枚导弹那个样子，下面 我都不这么罗索了。而且，在进一步复杂的分析中甚至可以说即有防空导弹又有 高射炮或者有战斗机什么的。

假设我军采用的是二次世界大战的老式的轰炸机。我之所以这么说，是为了 驳倒一些论点，就是好象对方的武器一先进，我方的武器一定要先进，否则不能 取胜的观点。那么，我军的轰炸机从进入敌人的导弹的射程范围内，再进入到可 以投弹的这段时间，需要五分钟。那么，在这五分钟里，敌人最多发射50枚防空 导弹。假设这50枚防空导弹的命中率为百分之百，即可以在五分钟内打下我军50 架飞机，而在我军进入可以投弹的距离时，需要10架轰炸机的投弹才可以将这个 导弹阵地炸平。这一点也是根据炸弹的爆炸力，准确度（由正态分布的方差决定）， 等因素通过一系列公式计算出来的。

那么，根据军事教科书的观点，我方必须出动60架轰炸机，在接近敌人时被 敌人击落50架，剩下10架将敌人的防空阵地摧毁。

我见过的我军的什么鱼雷艇战术教科书，什么火炮战术教科书，都是这样写 的。但这是错的，这不是毛泽东的战法，并没有构成绝对优势兵力。

因为，出动60架轰炸机，虽然一定能够摧毁敌人的阵地，但损失50架飞机。

那么，如果我是这60架飞机的飞行员，我会很不高兴，因为，我被击落的概 率是60分之50，接近于1 ，几乎是一个敢死队的打法。

而按毛泽东的思想，将60架飞机再乘上100 ，得6000架轰炸机，用6000架轰 炸机来摧毁这个导弹阵地。为什么这样呢？就是为了降低伤亡率。如果我是这6000 架轰炸机的飞行员之一，我不会感到非常地害怕，因为被击落的概率为6000分之 50，百分之一不到，我会认为也许我的运气没有那么差，正好是那百分之一中的 一个，因此我能够做到英勇作战不怕死。

当然，实际情况计算绝对优势兵力是要根据具体的武器，具体的射速，命中 公算等等计算的。但绝对优势兵力的形成与武器的好坏没有关系，与武器的数量 有关系。

有人提出清朝军队在鸦片战争中的失败，我也注意到了南斯拉夫对付北约空 袭时的失败，这两种失败都有一个共同的特征，就是自己的火力够不着对方。比 如说，自己的高射炮只能够打6000米高，而敌人的飞机在一万米高，这样敌人收 拾你，你没有办法。

因此，是否够得着才是重要的。比如敌人用B2轰炸机来轰炸，那么我方用战 斗机迎敌，在接近过程中我方会有20架飞机被敌人导弹击落，那么我就将这个数 字乘上100 ，用2000架飞机去击落敌一架B2轰炸机，这2000架飞机的飞行员还是 有安全的感觉。

而如果火力根本就够不着，当然只能被动挨打，无法取胜，可是这并不是我 军现在的武器的现状。我军现在的武器现状是连美国都够得着，因此不存在够不 着的问题。

而历史上清朝军队的火力够不着英军，如果采用正确的战术也不一定就会败。 主要是清朝政府害怕人民，搞政府抗战，不敢武装人民，而且不知道游击战打法。 实际上在够不着的情况下，就只能打游击战恐怖战了，攻台是不可能的了。

而毛泽东在抗日战争中由三万人马发展到100 万正规军，200 万民兵，其中 的主要武器是日本的三八大盖，是从日本手里通过游击战夺过来的，这说明了游 击战的可行性。

再回答有关二十万机帆船的问题。我的方案是这样，就是要问解放军在整个 解放台湾的战斗中用不用海军？用不用海军的驱逐舰？用不用海军的护卫舰，导 弹快舰，登陆舰这些东西？如果不打算用，那么我的二十万机帆船的战术也就不 要用。如果打算用，那么，是将所有这些海军舰船藏身在二十万机帆船中进攻好 呢？还是根本不进行这种藏身好？而我认为是前者更好，损失更少。因为，敌人 击中机帆船的概率，比击中舰船的概率要高一些，这样舰船就安全了一些。二十 万机帆船客观上构成了消极干扰。

此外，任何武器，哪怕非常简陋，也可以按毛泽东思想算出一个总攻的优势 兵力。比如在清朝，如果英军为三千，在步枪的射速为一秒钟一发，命中率为0.1 ，那么仍然可以计算出一个兵力数，向敌人发起冲锋时占绝对优势兵力。但算出来 的数一定非常巨大，估计要有三十万人同时冲锋，甚至一百万人同时冲锋。如果 没有这么多兵力，当然不能随便送死。如果是骑兵，那么数量会少一些，可能也 还是要十万骑兵同时冲锋。但如果在机关枪发明以后，敌人的射速增加，可能优 势兵力是天文数字，当然就不能贸然决战，但用游击战，不死守一个地方，可以 通过持久战的办法来消耗敌人的精力，但这就与攻台战役无关了。

因此，军事的原理并非一定要好武器。只要数量足够就行。一两件好武器决 定不了战争的命运。我在玩红色警戒时，曾经光用步兵，不用坦克，也还是可以 用人海战术占领敌人的阵地。

如果按有的人的想法，在人民解放军渡江战役前，应当是怎么分析解放军也 无法攻过长江的，见到解放军大造帆船，一定笑掉牙齿。但是数量是决定一切的。

因为事实上帆船造得太多，敌人就根本来不击将它们都击沉。事实的渡江战 役伤亡也并不是那么大。

总的原理是，计算兵力并不单单考虑能把对方拿下，而还要考虑增加兵力到 使自己的伤亡率降低，这样参加战斗的人员就会开心了。这样的兵力势必是非常 大的一个数目，美国实现不了这个，原则在于它征不到这么多兵，而中国可以征 得到，因此只有中国才能够适应这种战术。而钱，我已经说过，也并不是个问题。
