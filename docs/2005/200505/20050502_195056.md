从我对外企的观察来看国企改革的失误之处

[数学] 2000-12-04

我来深圳这十几年，有相当长的时期迷信资本主义的生产关系，因此也曾经 到外企工作过一段时间，并且也到国企合作工作过，也开发过国企的计算机信息 管理系统。我到过的外企是一个相当大的外企，IBM 公司投资的一个大的软件公 司。

但是，在我的对外企和国企的对比之中，我发现自己以前对资本主义制度的 管理方式的概念是错的。进而发现，直到现在许多人对于资本主义的管理方式的 理解也是错的，并发现中国这二十年来，国企的改革实际上犯了相当大的错误。

本文想对照外国大公司的管理经验来回过头看国企这二十年来改革有哪些失 误，有哪些本该不改却改了，而且改得一塌糊涂的地方。当然，认识真理要有一 个过程，但正如下棋一样，虽然前面下得不好可以原谅，但还是应当把前面已经 下过的且下得不好的地方进行复盘，研究一下本来怎样可以做得更好。

一开始中央采取的利改税的措施是错误的，如果按资本主义大跨国公司的做 法，是应当将所有的国企都合并成一个巨大的公司，所有的子公司上交利润，并 由总公司统一向中央财政交税。这并不会影响中国的改革开放进程，因为中国存 在着大量不属于国企的农民，有这些人参加进私企和外企，就足以使中国的经济 飞速发展。联产承包责任制的成功使人们产生了一个错误的看法，就是分散比集 中好。但从国外大的跨国公司的经验来看，集中比分散更具有竞争力。

即使是国企的饭店，由中央集中管理搞几个大的连锁店，象麦当劳那样的管 理，也还是能够挣到钱的。

发奖金的做法是错误的，国外大公司并不发奖金，只有小的公司到年终有分 红利的。大公司的管理策略是建立一整套完整的工资体系，建立等级制度，谁干 得好就有可能提拔。改革开放以前并没有奖金，只有工资，而且有一整套固定的 政策是正确的，只是需要调整各个等级的工资数额。而发奖金导致效益好的国企 资金多，效益差的国企奖金少，根据效益来决定企业员工的收入是短视的做法。 而国外的大的跨国公司并不将子公司的员工工资同子公司的盈利挂钩。比如说在 一个市内开了几家麦当劳，如果有一两家因为地点问题不景气甚至亏损，员工的 收入并不下降。从这个角度讲，倒是干好干坏一个样的。

而国内的发奖金导致一种混乱局面的出现，以至于工资体系呈现崩溃状态， 每个人的工资条上都有几十个项目，加上奖金的上不封顶政策，导致亏损的企业 和盈利的企业都拼命地发奖金。深圳市在改革初期国企的收入要高于私企的工资。

打烂社会主义大锅饭是错误的，实际上我观察的大的外企反而倒是大锅饭。

原来有一个错误印象是资本主义就是要解雇工人，但实际上大的外企一般不 解雇工人，即使是一个干得不好的人，通常的做法也是不给他加薪，但如果他厚 脸皮呆下来，肯定也能熬到退休的。比如香港电讯因为人员很多，有一段时间要 搞减员计划，也是说几年之内不再招收新的员工，导致一些人退休而自然减员。 日本战后有很长一段时间也是不解雇工人的。但外企的人员流动仍然很大，原因 在于一些人觉得前途暗淡，自己主动辞职。

承包的做法也是错误的，国外资本主义经济几乎就没有承包这个概念。也就 是说，中央为了发挥企业的积极性，搞权力下放的任何举措都是错误的。国企的 改革方向同农村的改革方向应当相反，并不是越来越分散，而应当越来越集中， 只有集中到中央统一管理，象大的跨国公司那样的管理，国企才有希望。

建立起现代的企业管理制度，就是要让每个国企的领导者权力不是很大而是 很小，小到报销一顿饭的钱都做不到。

我工作过的外企实际上官僚程序一点都不少，甚至买一个园珠笔报销起来都 极为麻烦，财务人员甚至要到文具商店去调查价格，各个部门经理通常也没有什 么权力。企业员工也并不都是在卖力干，甚至也是经常性地偷懒，企业的上下级 之间也是有大量的人治的成份，也还是要对上级点头哈腰地拍马屁升上去的机会 才大，也是有各种尔虞我诈的各种丑恶的现象。但是我发现，即使有着这些现象， 外企还是越办越大，公司还是在不断地润利，不断地发展，这是为什么呢？

首先是一整套成文的制度将各种丑恶现象约束在一定的范围内，使之不会对 公司的大的发展有任何大的害处，这里主要是死板的工资制度和升迁制度，还有 对工作量的监控。对工作量的监控是非常有效的管理方法，就是让每个人每天快 下班时都要填写一份单子，上面表示自己这一天干了一些什么。记得当时填这些 单子填得我烦死。这些单子输入到电脑中的数据库里，由公司上层管理人员进行 分析，对人的升迁的安排很大程度取决于这些数据的分析。

让大批人员下岗也是错误的，既然是劳动力，就可以利用他们的劳动来创造 价值，如果不会利用，只能是管理者无能的表现，组织起来的劳动当然比单个的 劳动效率高，对于弱势群体尤其是这样。而强人则根本不用你让他下岗，自己就 有可能被别人挖走。而个别管理者无能导致企业亏损无须过于着急。我注意到一 些外企在深圳头几年就是不挣钱的，他们也没有打算在头几年挣钱。

一些大的跨国公司也并不十分在意一两个子公司在一段时期内的亏损，这些 都是可以通过一系列手段进行调控的。而这种调控的前提在于中央必须有极大的 权力，而企业领导的权力应当很小。我参观过的香港电讯控制的香港的市话业务 就是长期亏损的，但是他们主要的盈利并不在这一方面，他们的头儿也并不过于 着急这件事儿。

对员工进行培训是大的外企的重要经验，我进的那家公司，一进去我就听了 半年的课，公司一点也不在乎我处于学习而非为他们生产利润的状态。我曾经问 过他们如果培训之后这个人辞职不干了公司岂不很亏，是不是要不许他走？

他们回答说如果一个人要走就最好让他赶快走，留下来反而是有害的。而这 种大规模的培训完全相当于赌博，如果有人培训完后就走就是没有办法，但大多 数人还是会接着干一阵的。而且培训的内容往往是这个公司的特定的办事程序， 其它的公司一般是用不着的。

请注意我这篇文章完全是从资本主义大公司的角度来看国企的改革方法的， 并不牵涉到什么社会主义的理念，完全是资本主义的东西，资本主义管理的经验。

我的观点是让全国的国企集中成为一家巨大的公司，由中央实行统一的管理， 死板的管理，对所有的厂长进行严格的权力限制，这样的大公司应当是盈利的。 这里需要的不是民主和分散，而是集中和专制。
