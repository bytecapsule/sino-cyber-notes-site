现在我认为法航事件确由种族歧视引起

[数学] 2001-05-15

事实上，法航将中国市长赶下飞机的新闻报导过程中，事实的真相大家是逐 渐清楚的。

首先我们看到的是新浪网上的一篇文章，说有一个中国市长非常地粗俗，素 质如何地低下。可是写此文的人也并没有目击到事件的全过程。因此，作为读者 的大家自然也就等待着进一步的消息，要看看中国方面怎么说，那位市长本人怎 么说。

当然，网上专门有一批崇美人士，一些崇洋迷外的人，或者一些对中国进行 和平演变试图打倒共产党的人，他们是不需要等真相出来的，在他们的头脑里只 要是外国警察对付了任何人，那一定是外国警察正确。因此，美国警察打死黑人 正确，法国警察赶中国人下飞机也正确。

这段时间除了有一个叫数学的混混拿了一元钱人民币在网上打赌说那肯定不 是市长外，其实中国人民是沉默的，他们在等待进一步的报道。而反华反共崇洋 迷外的人则是一片喧哮。当然，也包括羊城晚报的一位洋奴记者在不了解事实真 相的情况断定法国警察正确，甚至喊到“赶得有理赶得痛快”。

那么，在薄省长的另一面的讲话出来以后，现在我可以根据两面的意见进行 综合了。而我将两面的意见综合的结果，可以得出结论，就是法航在搞种族歧视。

事情的经过可以再说一遍。中国招商团的人上了飞机，市长坐商务舱而其它 官员坐经济舱，那么其它官员则将自己的行李放到市长的商务舱的行李舱，这样 就将行李舱占满了。那么后来的到商务舱的人行李就没有地方放，那么法国乘务 人员就将中国官员的行李拿走了一件，腾出地方来给其它乘客放行李。而这位官 员开始时不知道，后来才发现行李少了一件，丢了东西当然害怕，于是就急了。

因为，法国的小偷多也是举世闻名的。这样就和乘务人员发生争执，语言不 通，结果被乘务人员叫来警察把他赶下飞机。

整个过程相信正反两方都同意吧？

现在我来说明法航的人在搞种族歧视。

首先，将自己的行李放在飞行的哪个行李舱的哪个地方，这在世界各个航空 公司都没有明确规定。实际上是放在哪里都可以的。我记得有一次我乘飞机，先 上的飞机，把东西放好后坐下，后面来了两个外国人，一个人就指着我上面的行 李舱问：“is it empty ？" ，我这么臭的英语居然听懂了，于是说”yes"，结 果他们就将一个大箱子放在上面，随后再走进去很远后在里面坐下。这说明外国 人的观念中行李放在哪里都是可以的。

此外，在乘客上机过程中，乘务小姐应当一直站在那里等候的，否则就是玩 忽职守。而如果乘客有了什么不合适的行为，乘务小姐是应该当场制止的。比如 说，当那位官员和其它官员将行李放在了不该放的地方时，乘务小姐应当当场制 止，那么他们就会打消主意放到自己的坐位旁去。

再一点，行李舱通常是先来者先放，后来者后放。这是公平的，先来者当然 有先放的优惠。实际上航空公司都是喜欢乘客早到的。如果在飞机快起飞时才到 手忙脚乱，本来航空公司应当更加不高兴的。因此，中国乘客早早来到飞机上， 是守规矩的表现。而后来的人没有地方放东西，那么这是乘务小姐的责任，她们 必须给后来者找到放东西的地方。

那么，这里提到的种族歧视的地方就是，乘务小姐为什么不将后来者的行李 放到别处去？反正在到站之后会给送回来嘛。我认为后来者是法国人或者是白种 人，乘务小姐将给这样的人更多的便利。因为，放到别处去毕竟不方便。可是， 谁又让你后来呢？既然你后来，当然就会有一些不方便。可是乘务小姐不将后来 者的行李放到别处去，而将先到的华人的行李放到别处去，这就不公平，这就是 种族歧视。更何况在放到别处去时不和乘客打招呼，这就更不礼貌。

而那位市长发现东西丢了以后的表现和国内一样当然也不对。他的正确的作 法应当是叫警察来，说自己的东西被偷了，要求警察破案。结果事情却反过来了， 变成警察把他带下飞机了。

再说这件事在新浪网捅出以后，民族主义者和左派们都沉默着，中国人民也 都在沉默着，他们等待着进一步的报道，在没有彻底搞清楚真相之前不能乱发言， 以免象那个数学一样输掉一元钱。而那些个崇洋迷外的家伙们可就来劲了，一个 劲地发文说中国人民素质如何如何低下，共产党官员如何糟糕，因为他们的目的 就是要推翻共产党实现美国制度么，有了这么个事情怎么能够不发挥呢？

而在今天薄市长发言之后，这些人觉得光骂中国人素质不好会惹众怒，但是 乘机骂骂共产党还是有用处的。因此就启发式地问：“为什么这件事一出现网上 就一面倒地骂那个市长呢？”但事实上并不是一面倒，而是一群哈叭狗乱汪汪， 中国人民还沉默着呢。但照这个启发，似乎中国人民都是一听说市长二话不说都 要骂，因此，共产党的市长差劲，因此，要推翻共产党实现美国制度。

实际上中国人民是讲道理的，他们评价一个市长并不是二话不说就乱骂，他 们要看这个市长具体做了什么。只有一些帝国主义养的哈叭狗们才会一听说是共 产党的市长就乱骂，一听说是外国警察赶了中国人下飞机就一定是中国人不对。
