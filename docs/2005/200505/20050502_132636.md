成熟的制度获得成功说明它有潜力

[数学] 2001-01-30

许多人认为毛泽东管理中华人民共和国的前三十年经济上是失败的。我原来 也这么认为。但现在看来，毛泽东是成功的。

我曾经喜欢玩过一种电脑游戏叫“红色警戒”，抛开其中对社会主义国家的 丑化和对美国制度的赞颂不谈，其中的玩法是很有意思的。每一场游戏一开始你 只有非常少量的部队和工事，有一些钱，这些钱你可以用来造坦克，训练士兵， 造新的工事，然后向敌人发动进攻，但如果这样你的钱很快就花完了，最后眼睁 睁看着部队被敌人打垮了。因此正确的玩法是一开始要造炼矿厂，造矿车，这样 就可以开矿，就可以挣到许多钱。而生产能力一增加，钱一多，什么都好办了， 可以生产出大量的坦克和士兵，即使战略上错一些也没有什么问题了。这种游戏 有几个级别，其中最困难的级别，是一开始你只有很少的钱，但敌人向你不断发 动进攻让你有很强的欲望要造坦克和士兵来防守。可是这么一来你就没有钱造炼 矿厂了，经济一枯竭最后还是被打垮。因此就要想办法用最少的部队来和敌人抗 衡，还是用尽可能多的钱投入生产，建立较多的炼矿厂，经过一段非常困难的时 期之后，生产能力大大增加，这时候情况改观，钱已经哗哗直长，胜利就是必然 的了。

而中华人民共和国的前三十年，正是这种情况。发展起点，钱没有钱，大部 分人民都是文盲，工业基础一点没有，什么都不会生产，又遭到帝国主义的封锁 和打压。在这种情况下完全凭自力更生创造出了一个工业基础，完全是一种奇迹。

攻击这个时期的人之所以能够奏效，主要是攻击这个时期人民的生活改善得 很慢。

虽然人均寿命增加了十岁，人口也由五亿发展到了八亿，普及了教育，实行 了大规模的疫苗注射，但生活确实改善得很慢。

再打另一个比方，假设造房子只需要砖（一种抽象），而要生产砖就需要砖 窑，而生产砖窑又必须用砖。那么，为了提高生产砖乃至生产房子的速度，第一 步就是将生产出来的砖大部分用来生产新的砖窑，而新的砖窑生产出更多的砖， 这些砖再大部分造新的砖窑，这就使生产能力以几何级数高速增长。这就是当时 的计划经济的特征。但这样的计划经济如果没有到一个停止的时候，就形成了异 化，就是生产能力已经大大提高之后，人们总还是没有房子住。因此要一个转型， 在生产能力提高到一定程度之后，生产新的砖窑的行为就开始停止，转而大量地 生产新的房子。

因此，共和国的前三十年主要靠勒紧裤带搞工业基础，工业基础一搞起来以 后，转型为市场经济，人民生活水平就开始迅速提高。之所以一改革开放人民迅 速富起来，和中国已经形成生产规模有关。如果什么都不会做，卖不出什么东西， 这钱又怎么能挣来，象现在美国到处都是“中国制造”的产品，说明中国造得出 来，为什么不是巴西制造或者印度制造？造不出来东西，拿什么和别人做生意？

记得中国历史上曾经有过八王之乱，说是皇帝荒淫不了解下情，当下面报告 说百姓们遭灾没有饭吃时，奇怪地问道“为什么不吃肉”。而现在的一些人评价 前三十年，也是这么个评价，“为什么那时候不搞市场经济？”。而我认为，一 个国家如果要发展成中国今天的规模，就必须要经过一段计划经济时期，要有一 个勒紧裤带，全体人民同心同德地搞积累的时期。现在看来，那些个一直没有搞 过计划经济的那些国家，如亚非拉美的广大发展中国家，即使一直在努力地学习 美国的制度，但和美国的贫富差距是越来越大，债台高筑。

从实践是检验真理的唯一标准来看，美国制度是一种合理的制度，但却是一 种非常难学的制度。可以说现在还没有一个发展中国家，因为学习了美国的制度 后就变成现代国家的，一些小地区可能发财，但不具备美国的技术能力和生产能 力。而亚洲四小龙本来可以当作成功的例子，一个亚洲金融危机又打趴下了。亚 洲四小龙成功的时候却是不太按美国标准来的时候，那时候这四小龙都即没有多 党制也没有大选。而且，他们的发财也象是玩游戏中捡到了工具箱，一下子得了 一大笔钱，却没有形成技术能力和生产能力。

而如果一个制度非常难学，学习他的人怎么努力也学不好，改善不了状况， 本身就是这个制度的一个缺点。比如菲律宾，什么美国的制度都学了，历史上也 是美国的殖民地，现在又有大选又有多党制，也有市场经济。可是就是一个乱七 八糟的地方，人民选举总统，不是选演员就是选美女，结果政治上一塌糊涂。

因此，中国的发展过程给广大发展中国家提供了一个典范，那就是，先建立 一个集权政府，在全体人民中建立统一的信仰，经过一段勒紧裤带的时期建立一 套工业基础和技术基础，随后再转型为市场经济。

有一种错误的“共识”，就是说计划经济培养懒汉，这是不对的。实际上毛 泽东时代的人民是干劲冲天的。从对工作的精益求精的态度可以看出这一点。

怎么看呢？我们知道一个懒汉做的事情一定不象样子，看着就不象一个活儿。

而勤快的精益求精的人做的活儿就象一活。我们现在很少能够看到前三十年 生产的产品了，它们大多旧了。但有一种产品今天我们还可以看到，那就是前三 十年出的书。我不是说书的内容，而是说书的印刷质量，其中排版是不是认真， 中间有没有错字白字。那么如果我们看共和国五十年以来的各种书籍的印刷质量， 可以认为质量是一个下降的过程。以前的书由排字工人排，但极难找到错字白字， 其中的数学公式很难找到错，从那些书就可以看出当时的工人们有着怎样的极大 的精益求精的精神。而现在生产手段已经成了电子排版了，可以做到豪华，精美， 但就是做不到错误少，一看就知道都是一些粗枝大叶的人搞出来的。

再说电影，五十年代的电影技术水平是不如现在，但从那些电影中演员们的 认真劲，每一个镜头都精心制作，可以知道那时候的演员们对演技是精益求精的， 虽然并没有多少收入，也没有什么拍广告，也没有大腕，可是人们就是很认真。

如象“南征北战”这样的片子，其中的解放军，无论是干部战士，是指挥员， 还是敌人，还是老百姓，都演得非常逼真。而现在的所有拍解放战争的片子，中 间的演员，一看那个作派，马上就觉得不是那个时代的人，演得马马虎虎，他们 就缺少这种认真劲儿。现在的演员们演什么都给人一种假的感觉，感到他们做事 情不认真。

以前的学生认真还是现在的学生认真？我认为还是以前的学生认真。以前的 学生看书探求真理的态度可以说是令人感动，每一个问题非要弄明白不可。而现 在的学生则非常地不努力，粗枝大叶得很。我记得七七级上大学的那些人，对知 识有一种强烈的饥渴感，记得当时有一门大家可听可不听的课，就是说你来也可 以不来也可以，即不影响毕业也与考研究生无关，叫做“数字信号处理”，结果 大家都知道这门课重要，当时一个大教室全部坐满后过道和后面还加满了椅子。

这是七七级的事。到七八级就只有三十人听这门课，七九级就只有三个人， 只好将它去掉了。

搞市场经济的人有一个重要的推论的前提，许多复杂的数学公式就是从这个 前提推导出来的，这个推论的前提就是“经济人”的概念，也就是说，将社会上 的所有人都看作自私自利的，追求个人利益最大化的人，换句话说就是“人不为 己天诛地灭”。因此搞市场经济的人对于计划经济时代的人们的干劲是不理解的。

当我提出“信仰制约”的时候，他们就都认为是一种笑话。

但信仰制约是一种客观的存在。

任何制度都是先由一些建立了信仰的人，形成组织，再建立制度，并由信仰 来维持制度。中华人民共和国的制度是这样建立的，一开始是由一些信仰共产主 义的人，按照这个理想，努力奋斗，通过暴力手段打碎旧的管理，建立新的管理。

美国的制度也是这样建立的，是由十三个殖民地的人，信仰了某种真理，打 碎英国的管理，起草了独立宣言，采用暴力手段夺取政权，建立美国制度，并由 一些信仰这个制度的人来维护这个制度。在南北战争时期，美国面临分裂的危险， 那么仍然是一些信仰美国制度的人经过战斗维护了美国的统一。如果那些北军的 战士们个个都追求个人利益最大化，那么为什么要统一呢？完全可以分成两个国 家单过，还少流那么多的血。

因此，有信仰在先，有制度在后，并通过信仰来维护制度，这就是人类所有 稳定的政治制度的基本模式。也就是说信仰制约高于制度制约。
