美国社会与文化，观察与思考（26）――美国的黑人问题 

美国黑人占美国人口的百分之十二左右，即有三千五百多万人。美国社会的黑人问题是一个相当复杂非常棘手的问题。就是说，这不完全是一个一清二白的正义与邪恶、压迫与被压迫的问题，而是各种原因缠绕在一起，包括黑人群体内在的各种因素的相当复杂的问题。 

当然，美国的黑人问题确实起始于邪恶与正义、压迫与被压迫，黑白分明一清二楚。哥伦布发现美洲“新大陆”以后，从欧洲来的白人就把这印弟安人民世代居住的家园当作上帝奖赏给他们的乐土。美国的南方地势平坦、气候湿润炎热，很适合大规模的农业种植，缺乏的是大批在农田里干活卖命的农业工人。欧洲白人用大船从非洲运来大批黑人做奴隶，在南方白人的农场里种棉花、种其它能赚钱的农作物。黑人们还包下了白人农场主的所有家务活：清理、修理、缝衣、洗衣、做饭、照顾孩子。黑人没有任何的人身自由，任打任杀，像一件工具一样完全属于白人农场主。 

我曾经不解地问过若干美国人民：为什么你们的祖先舍近求远，不强迫美洲印弟安人做奴隶，却要跑到非洲抢来黑人做奴隶呢？几位美国人民瞪眼摇头，坦白地说：不知道，从来没想过。只有一个明白人说了：印弟安人的文化风俗是天然的我行我素，他们不会、也不能适应完全的俯首听命做奴隶，而且他们的体力也不能适应长时间高强度的农田劳动。明白人接着说，有的白人农场主是强迫印弟安人一天接一天从早到晚地在田间劳动，但是没有多久这些印弟安人就全死去了。 

到十九世纪中期，美国北方的工业已经发展得很像样子了，是先进的资本主义生产关系了。但是南方还是农场主为奴隶主、黑人为奴隶的奴隶社会，比封建社会还要落后一大截。美国的南北战争可以说就是北方新生的资产阶级与南方奴隶主阶级之间的战争，结果当然是能造枪支和大炮的先进的北方资产阶级赢了。林肯总统宣布废除奴隶制（Emancipation Proclamation），南方的黑人农业奴隶的解放为美国的工业大发展提供了大量所需的劳动力。 

但是黑人仍然是美国社会的另类，他们几乎没有作为美国公民的所应该具有的任何权利（北方要好一些）。美国社会是一个种族隔离黑白分明的社会：黑人生活在社会的最底层，做着最脏、最累、最苦的工作，黑人不能进入白人的学校、白人的餐馆、娱乐场所，乘坐公共汽车黑人必须坐在车厢的最后。而且黑人当然没有政治选举权。促使黑人的社会地位有了根本性变化的还是二十世纪五十年代之后的民权运动（civil rights movement）。为了给黑人争取同等的政治权利和真正的平等，美国开明进步的人权民权主义者们（包括黑人和白人）与广大黑人群众共同努力奋斗几十年，同时随着社会的进步，终于从六十年代起形成了一股声势浩大的民主人权运动。约翰逊政府的“伟大社会”运动（Great Society），建立起了美国现在的很多社会福利制度，如老年人基本免费医疗保险的MediCare，残疾人的福利，失业救济金，贫困救济金等等；同时，政府制定具体措施，最后确定了黑人的选举权，以及其它有关的各种平等权利。 

虽然在南北战争后从法律上说，黑人有政治选举权，但是在实际上却是一句空话。尤其是南方的各州，白人的政府以各种方式阻止黑人的平等权利。南方的黑人在六十年代最终取得了选举权。南方各州的公立学校取消种族隔离制度，黑人学生与白人学生开始同班上课。美国政府为了保证黑人的平等权利，制定了反种族歧视的法律，制订一些强制性的政策，例如：各级政府机关必须雇用一定百分比的黑人雇员、各州的公立的大学必须招收一定比例的黑人学生，这项政策称为Affirmative Action。任何雇主必须同等对待黑人和白人雇员；任何公共场所必须将黑人和白人顾客、观众同样同等对待。否则，黑人可以根据反种族歧视的法律起诉这些政府机构、学校、雇主、饭馆和娱乐场所等等。 

八十年代初我来到美国（北方）读研究生，一眼看上去，黑人白人是平等的，没有明显的种族歧视现象。但是住了一两年后，渐渐看到的实际问题却是很深刻而且严重的：白人纷纷搬到郊区新建的环境好的地方居住，把城市里设施落后拥挤的市区留给黑人居住。一个白人居民区，若有黑人搬入，房地产的价值马上降下来。黑人居住区比较破旧，很多黑人没有工作，他们就依靠政府发放的各种救济金生活。黑人居住区的犯罪率是最高的：吸毒、贩卖毒品、暴力犯罪；由于工作机会不好找，很多黑人青少年游手好闲不务正业，走上吸毒、贩毒、加入有组织的犯罪集团的道路。 

这里必须提一下黑人的传统文化问题。这是一个不能公开讨论的非常敏感的所谓政治上不正确的问题（politically incorrectness）。但是要想弄清美国现在的黑人问题，这是一个不能不涉及的根本性的问题。因为例如美国的亚洲移民从一定意义上来说，社会政治地位不如黑人高（美国政府的Affirmative Action 政策是为黑人订立的），但是美国的亚裔群体（当然有自身的问题）就不存在黑人群体的诸多令人头疼脑袋大的问题。 

非洲的黑人本来自己好好地呆在非洲的丛林里、原野上，过着自己的或打猎、或采集、或原始种植的自由自在的、以部落为单位的生活。当然他们也有部落之间的战争、死人、流血，但是打完之后，还得自己去找吃的。而且由于没有现代化的枪和炮，打仗也死不了那么多的人，也不会造成那么大的灾难。非洲黑人们热情豪放，擅长音乐和舞蹈，一遇高兴，全部落或几个部落就聚会在一起，敲起鼓、画上各式脸谱、男女老少共同载歌载舞，尽情快乐。这是为什么黑人具有音乐天才、舞蹈天才、表演天才，以及体育天才的基因了。 

非洲黑人的部族社会基本不是农业社会，更不是工业社会，所以他们也就没有与农业社会相适应的文化风俗，例如一夫一妻家庭的概念、自我约束的能力、个人的责任感，等等。由于没有很强的像亚洲人那样的家庭观念，黑人的男人并没有强烈的当父亲、作为一个家庭的家长的责任感，他们可以与不同的妇女发生性关系，然后拍拍屁股就走再去找别的女人，妇女生了孩子与孩子的父亲没有太大关系。于是在美国的黑人群体中，造成一个祖母带着几个女儿，以及一大群孙子孙女作为一个家庭，父亲、祖父不知跑到哪里去了，全家人靠着政府的救济生活的景象。由于家庭中缺乏父亲的正面形象，有的母亲忙于生计，有的母亲不务正业，孩子们自由放任、为所欲为，学坏的为多数，很多还未成年就进出少年罪犯管教所若干次。美国很多大中城市的内市区已经基本都成为了黑人居住区，黑人居住区是典型的犯罪严重的地区。一些城市的政府为黑人建造了经济的居民住宅楼，但是很多黑人居民不知爱惜（他们以政府的救济金付租金），却是拆了楼房的管道、电线拿去卖钱，把居民楼弄成贩卖毒品、集团犯罪的基地。很快，居民楼就变成无法住人的废墟。 

八十年代末我到南方教书之后，更是感到了美国黑人问题的沉重，沉痛，和说不出的不是滋味。 

居住在南方的黑人比居住在北方的黑人要多。南方白人与黑人之间的隔阂是根深蒂固的。很多白人至今认为十九世纪的解放黑人奴隶的南北战争是错误的。白人落后顽固的偏见倒是不难理解的，但是让人痛心、不容易理解的是黑人的一些不争气的行动和认识。 

应该说，自六十年代的民权运动起，政府制定了保护黑人的各种政策与法律，带动起来美国各界人士的积极行动，到八十、九十年代，美国全国上下形成了一种舆论压力：不能在任何公共场合主流媒体上批评指责黑人，哪怕是黑人做错了，也不能说。这大概就是矫枉没有办法不过正了。这叫做“政治上的正确性（politically correctness）”。 

有一些黑人把现今美国黑人群体存在的一切问题，都归结为由于白人在历史上对黑人的剥削压迫和不公正不公平的对待而产生的。黑人中明智有头脑的领袖们，认为要改变现在美国社会的黑人问题，主要是依靠黑人自己，一定要首先提高黑人，尤其是男性黑人自身的责任感，提高他们对自己、对家庭、对社会的责任感。黑人的领袖们曾组织过号称百万男人（实际是几十万）在首都华盛顿的大集会，希望促进提高黑人群众的自身责任感。但是作用不大。美国监狱中的男性犯人，百分之五十以上是黑人。这与百分之十二的黑人人口比例相距甚远。 

政府规定的要照顾黑人的Affirmative Action 政策的实行，对于改变美国黑人的社会地位确实起到了很大的作用。当然也产生了不少问题。为了照顾黑人，面对同样的招工、招生条件，不少合格的白人申请者被不合格的黑人申请者挤了下来。贫穷白人的社会地位往往比黑人还要低，他们成为社会上没有人关注的人群。 

再说几句有关Affirmative Action 这个政策的话。Affirmative Action这个政策本身是违背公平竞争原则的，这个政策目的在于照顾黑人，改变黑人的社会地位，给黑人群众一个有所保证的起始出发点。我认为这个政策是正确的，而且对于改变美国黑人的社会地位是必须的。但是这个政策的实行也有它的无法避免的副作用。一是损害了一部分白人，尤其是社会下层贫穷白人的利益，加深了下层白人与黑人的矛盾。二是这个政策的实行，在一定程度上助长了很多黑人的依赖心理：他们很是缺乏自强不息、自力更生的精神与气质，有了的问题不从自身找原因、找动力，总是抱怨他人，等待政府和别人来解救、解决和救济。 

另一方面，得利于政府的这个Affirmative Action 政策的少数黑人成功人士，当他们进入美国社会上层之后，几乎全都成为共和党保守派，成为白人极右派的走狗，在与黑人利益有关的政策问题上，他们坚定地与保守派共和党站在一起，他们打着公平竞争靠自己的旗号，主张取消Affirmative Action这个政策，完全忘记了自己当初正是靠着这个政策的执行而成功、从下层爬到了上层来的。现在美国社会上下有一股共和党保守派煽动起来的舆论，认为现在美国黑人的社会地位已经很不错了，应该是取消政府的Affirmative Action 政策的时候了。我是认为，虽然这个政策有着这样那样的问题，但是还是应该执行下去，因为不这样的话，问题会是更大更麻烦。 

我认识的一位美国朋友在一个大卡车的拖车修理厂工作。这位朋友为人正直、友好，很有同情心，敬业精神很强。他多次说到他们车间里的黑人工人，每次说起来都是摇头不止。有极少数几个认真负责勤恳踏实很好的黑人工人，但是大多数黑人工人的敬业精神很差，非常缺乏耐心，干什么活都没有持久性。我的这位朋友几次帮助新来的黑人工人学技术，试图帮助他们培养起认真勤恳的劳动态度，但是不敢严格要求他们，又不能当面指出他们的毛病（这是政治上正确性的社会压力了）。所以大多数黑人工人工作起来吊儿郎当，技术上靠不住，车间和工厂的头儿没人敢说，又不能解雇。这些黑人工人楞是成为令人无话可说、只能摇头不止的不能扔掉的烫手山药、 

我们学校里有不少黑人学生。他们大多对人友好，容易相处。我是教数学的。通过教学，我感到黑人学生们的一个大问题是基础数学知识差、基本数学训练几乎没有。白人学生中也有基础很差的，但是也有很多相当不错的。而黑人学生是普遍的基础差。但是大巫还在后面，真正使我惊奇不已的是：尽管黑人学生们的数学基础和基本训练是如此的差，但是他们却是普遍的自信心十足、雄心野心大得出奇，或者可以说是大得近乎狂妄了。我问过不少学习基础数学课程很有困难的黑人学生，以后打算选学哪个专业，比例不小的学生回答说：要上医学院学医（在美国是要大学毕业后，通过极为严格的统一考试，才能进入医学院）。惊奇得我下巴也快掉下来了：连高中数学都弄不懂，怎么可能通过那个极为严格的进医学院的资格考试呢？但是这些学生们可是没有开玩笑，他们很认真。 

这不由得我不纳闷很惊奇。到处搜索询问了很长一段时间以后，我终于弄明白了：从黑人孩子们上小学起，就没有教师敢于严格要求他们，尤其是白人教师――稍微管严一点，学生或是家长告你一个种族歧视的罪名，这可是能让你进监狱的。就是黑人教师，面对着一大堆自由、独立、人权的社会压力，也是畏萎缩缩，得过且过，任学生们自由行事，爱学不学。老师们对学生们，尤其是对黑人学生们只能是又哄又捧又许愿：你将来可以做你想做的任何事情，只要你有愿望，就一定能实现。小学教师把黑人学生连哄带拍地捧出小学，送进中学；中学教师又战战兢兢地许着愿把他们捧出中学（而不论他们有没有达到中学毕业的水平），进入大学。大学本来是应该根据学生们的SAT分数来录取的（SAT是对中学毕业生的基本知识和学习能力的一种考试），但是因为有了政府要求的一定要招收一定比例的黑人学生的Affirmative Action政策的规定，各大学都要降低标准，招收一定比例的黑人学生。于是很多黑人学生被捧了一路，直到进了大学也始终不知道自己的真实水平是什么。这就是我面对着的雄心、野心勃勃，但是基础知识和基本训练差得一塌糊涂的黑人学生们。 

几年前，我的基础数学课（基本是补习高中的数学）上有一位黑人女学生，她告诉我说，她毕业后要进杜克大学（一所很有名的私立大学）的医学院学医。我说，要进医学院你得首先通过医学院的统一考试呀。她说：不用，因为杜克医学院专门有一个培养黑人医生的项目，她已经与掌管这个项目的有关人员联系好了，只等从大学一毕业，就进杜克的医学院。这个女学生倒是很用功，但是就是能力不行，或者用俗话说就是：真不是学习数学的料，不论我怎么讲、如何解释，她就是弄不懂。这个学生在其它的几门课（化学、生物）上，也有同样的问题，这几门课的期中考试不及格。这个学生跑到校长那儿，告了一状，倒打一耙，说考试太难了。面对着黑人学生，校长也没辙，通过系主任悄悄地传下话来：请求大家想办法帮助这位黑人女学生过关吧。于是我只能想花招开小灶帮助这位女学生过关（不是帮助她学习，因为她的能力和水平明摆在那儿）：考试不及格，我给你补考，还是这几道题，只是把数字换了一换。补考之前，我认真反复给她讲解怎样解每一道题的具体步骤：记住第一步做什么，第二步做什么，...。终于，她以最低的可能成绩通过了这门课，我也可以坐下来好好喘口气了。 

通过教数学课,我确实看到不同的人学习数学的能力是不同的。对于能力实在差的白人学生，我跟他们谈话聊天，问他们打算学什么专业，一般我能够直接地劝告他们在选择自己的专业时，不要选与数学有关的学科专业，例如各类自然科学、计算机科学，等等。而对于同样数学能力差的黑人学生，我从来不能直来直去地说话劝告，最多是拐着弯抹着角地跟他们说：应该多看看你自己喜欢什么专业，而且你能适应那个专业，你个人的能力可以让你在那个专业上干得好活得自在。 

我很喜欢绝大多数的黑人学生，他们友好、真诚。但是对于他们的前途，我感到相当的不乐观。 

我有一种看法（这样的看法是不应该公开讨论的，好在这里是网络论坛），黑人的文化不适应资本主义市场经济社会。这是众多美国社会黑人问题产生的内部原因。我常想：如果白人没去非洲把黑人当奴隶运到美洲，如果世界上的其他人类文明（欧洲、亚洲）不去打扰非洲人民，让非洲人民按自己的意愿文化风俗习惯，按照自己的生活方式自由自在地生活在自己的土地上，那又会是怎么样呢？ 

这么说吧，为什么资本主义市场经济的现代社会，就一定必须是不同人类文明都必须努力奋斗玩命争取的目标呢？
