江南・河南  

[语迟] 

江南这个题历代文人写得很多。比如写西湖的，不计其数。写了这么多，后来还是一个日本人，哪个朝代的事情忘了，写了一首诗不错： 

昔年曾见此湖图  　　不信人间有此湖  　　如今打从湖上过  　　画工还欠费工夫 

就是这样。  　　中学课本上有宗璞的写西湖的文章，现在不知道还有没有，我认为写得那叫一不好，印象全无。倒是有一篇写苏州园林的科普文章，老老实实地写四大园林怎么布局，怎么怎么，还让当时我们这些孩子挺向往，当时我们在新疆，苏杭实在太远。那时侯都还不富裕，同学中间只有一个去过苏杭旅游的，老师就叫了他在台上讲，讲得我认为比宗璞写的要好很多，讲的事情我现在还记得。下江南的时候还按着他讲的去找，也找到了。老师自己是杭州人，我认为她讲得就不行，太激动影响了发挥，说起故乡，几乎要晕倒。老师还拿了很多她在故乡的照片来散发，照片这个东西，很多情况下只会倒人胃口。有那么一种东西是照片、录象拍不出来的对吧。  　　总之，历代文人小资写江南写了无数文字，我认为无非就是要写一种东西，而这个东西又好象写是写不出来的。 

我第一次去江南是96年的初春，应该是大三的时候，是母亲带我去的。第一站是无锡，接着是苏州、上海、杭州。带我去了一次，我当年自己又偷偷去了两次，扬州同里周庄什么的，后来又去了几次。我深刻体味到了那种写不出来的东西。这个东西对人的作用，就是心中的一阵小抽搐。这几乎是个生理反应。当然说抽搐是有些不合适的，因为不是那么剧烈难受，反正就是麻麻的，酥酥的，挺忧伤。比如想起了过去一个美丽忧郁的同学，中学时候那还是很纯情的嘛，不象现在已经是标准老男人了。 

西汉时期，江南还是很菜的，司马迁记载江南，说这个地方低洼潮湿，物产丰富，江南人特懒，几乎不事生产，随便捞几条鱼捡点野菜稻子往地里一扔也不管，就可以活得有滋有味。所以这个地方穷的人不多，富的人也不多。当时全天下羡慕向往的灿烂福地是长安京畿地区，汉中一带，还有中原河南。

到了东晋南北朝的时候，北人南迁，江南这才进入大规模建设时期。这个“北人”，主要就是河南人。东汉京都洛阳，曹魏在许昌，西晋京都也在洛阳。后来是“五胡乱华”，其实当时是十一胡乱华，为什么说是五胡乱华？就因为只这五个胡专门捡河南中原这个心脏地带乱，这么一乱，河南土著能跑的全跑了。

从西晋开始，中国人开始吃败仗，西晋是中国历史上第一个被外族灭亡的朝代。河南土著跟着中央跑，东晋建都南京，河南人就开始在苏南地区泛滥。后来很巧合的是，唐末乱华的也是“五”代，也专门捡长安－洛阳一线乱，河南人又只好跑路。这一次，跑得远了，沿着东南沿海跑，跑到浙江，一直跑到福建、广东。

再后来是南宋，河南人这次跟着中央跑到杭州，开始在东南沿海大规模泛滥。这就是后来的客家人。所以现在的广东话，粤语，就是唐代的河南话，也是当时的官话，普通话。闽南话就是宋代的河南话普通话。不要不信，这是真的。

下面这一部分是语迟首创，我没有在别的地方见过这样的说法。响应数学号召，诸位也帮我记着，以后看到类似说法，那基本上可以肯定是从语迟这里来的。如果谁发现以前也有类似的说法，那是英雄所见略同。不过可能性不大。

中华文明其实由三个部分组成：中原文明、湘楚文明，还有巴蜀文明。其中湘楚激进、巴蜀保守、中原介于两者之间。

比如湖南这个地方，70%的山，20%的水，只有10%的土地。此地人民只有发奋图强劳作不辍，才能生存，活着艰难，就要多谋善断，积极进取。这是湘楚文明之所以激进的原因。

相反四川天府之国，物产极其丰富，风调雨顺，战略上又易守难攻，是中国人的一个安乐窝。所以川人相对懒蛋，贪图安逸享乐，不出川成不了气候。

中原这个地方就很有意思，虽然土地肥沃物产丰富，但是必须赶农时，24节气跟鞭子一样赶着人努力，否则错个几天就可能颗粒无收等着挨饿。所以河南人就介于激进与保守之间，中一个庸看。

此三者几乎覆盖整个中国，鲜有能出此范围者。比如上文说了，河南、陕西、山西、山东、安徽、江浙及东南沿海各省，这属于中原文明。两湖、江西一带是湘楚文明。四川、重庆是巴蜀文明，剩下的就是交错地带，比如云贵、广西等等。

河南是中国人的老窝，是个染缸，什么族的人掉到这个染缸里一染，就变了样。而且再也难变回来了。从五胡乱华开始，一波又一波异族在这个染缸里染过，变了样，再扩散到其他地方，接着染其他的族类，染到今天，就是中国。几个不是河南人？！其中还有犹太老乡，公元十一世纪，犹太民族的一部分迁徙到京都开封一带，被称为“挑筋教”，染了这么多年，也找不着了（当然后来还有，又来了几批，以色列一直希望中国能够将中国的犹太人列为少数民族之一）。西方公认的桀骜不逊无法同化的犹太民族，掉到这个染缸里，照样。 

所以现在河南本省的人，却不是华夏正裔。最正宗的华夏后代，应该在江南。更具体地说，应该是苏南。镇江，南京，无锡，常州，苏州，这些地方。江南，去过的人不知是否注意，分为两种。我在昆山下面的周庄就很明显地发现当地土著似乎非我族类。而与周庄半小时水路（快艇）的苏州下面的同里，人就完全不一样。比如杭州人的北方色彩就比较浓厚，大嗓门，口音很硬，相比吴侬软语难听得很。就是说，如果细心一些的话，会发现江南人中南人北人直到现在，还有分界。  　　而河南这个地方，有次看新闻洛阳哪个地方抓了一个假钞贩子头目，某某乡某某村二流子，40多岁，穿得破破烂烂的被架着膀子拍照。我一看，他妈的，怎么是个欧洲人？  　　古罗马十字军东征的时候，遭遇强大匈奴，曾经几千部队被匈奴整体俘获改编，再与汉朝打仗，又被囫囵俘虏。后来朝廷将这几千欧洲人安置在甘肃安息一带，繁衍生息，直至现在。甘肃安息靠近新疆，兰新公路路过这个地方，当年家父就曾经目睹这些罗马人的后代，纯粹的欧洲人的面孔。罗马面孔与斯拉夫人波斯伊斯兰面孔的区别是很明显的。而在今天河南这个地方，仔细留心的话，这些面孔都能发现。河南，是个世界民族的交汇地区。 

这其实是非常有意思的事情。根据《汉书》记载，当年我部队在一次作战中发现匈奴军队中首次出现了奇怪的一群，盾牌长矛铠甲，跟鱼鳞一样，名为“鱼鳞阵”。就全抓了起来看个究竟。这就是那些倒霉透顶的罗马人。93年左右新疆爆炒这件事，想与甘肃合作搞个旅游项目的意思。当时发的照片，是安息县的贫困小学的可怜娃娃们，破破烂烂的教室里，一屋子破破烂烂的欧洲孩子，看得人感情复杂。文章还记录了身高1米90金发碧眼的甘肃农民，操着一口当地土话对乡镇干部表示不同意见的情景。  　　其实这种现象在河南农村，是经常可以发现的，不过较分散而已不如安息县那么集中罢了。比如唐朝东都洛阳当年就曾有20万波斯人定居，这些人的后代都去了哪里了呢？很多都在当地湮没了。 

诸位不要误会，我不是要说现在河南人名声臭啊臭啊原来都是些老外。不是不是，不能这样误会。  　　上文不是说了吗？河南是个染缸，一方水土养一方人，无论哪里的人，哪个民族，到了河南，沐浴着四季分明温和适宜的气候，吃着小麦，喝着黄河的水，不知不觉的，就变了。那些至今外貌上仍然保留着祖先痕迹的人，也不过就是面貌上还有些祖先的痕迹罢了。即便就是江南那些华夏正裔，１千多年吃米饭喝着长江水，原始的族性也改变了很多了。 

结语：  　　所谓的中国人，是一个民族融合的结果。这个民族融合的过程，从黄帝大战蚩尤开始，一直到今天，都在继续。正因为无比包容的中国文化使得民族融合毫无障碍，今天的中国人，可以说是凝聚了无数民族优秀基因的最优秀者。这同样是一个判断的标准，哪一个民族曾经容纳包容的其他民族越多，哪一个民族在人种上就越优秀，反之，故步自封，种族主义，盲目排外，自私狭隘，那是没有前途滴。
