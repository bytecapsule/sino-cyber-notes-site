从列宁到特朗普 人类钟摆的百年回拨

王陶陶  陶太郎

2017年1曰20日，唐纳德特朗普就任美国第45任总统，很少有人能够清醒地意识到特朗普总统对于世界究竟意味着什么，就像100年前很少有人能够理解俄国的列宁革命对于世界的后续冲击。

舆论往往高估政治领袖的气质和教育水平所起的作用，但历史证明，这两种素养在政治斗争中并不重要。事实上，政治家最重要的品质，在于他的洞察力和政治意志——这也是舆论低估了列宁、斯大林、毛泽东的重要原因——特朗普胜选后，舆论普遍认为胜选关键在于他的美貌女儿伊万卡

因为，美国新总统特朗普与100年前的布尔什维克巨人列宁，实际上有着相当惊人的相似之处。

1、他们都曾被认为是敌对国家的政治工具。

1917年3曰27日，列宁通过德国人（当时正与俄罗斯交战）的帮助，乘坐“铅封列车”，取道德国、瑞典回到俄罗斯，并且在德国军方的资助下，推动自己的布尔什维克革命——列宁也由此被认为是德国人的间谍；特朗普在竞选过程中同样得到了俄罗斯舆论和情报部门（涉嫌）的帮助，并被部分美国舆论认为是普京的玩偶。

2、他们都深切意识到群众运动在现代政治中的决定性力量，并热衷于通过与民众直接交流来把握政治潮流的动向，且成功地赢取了基层党员的热情和支持。

“此人不会是官僚，因为官僚僵化的思维使他无法融入大众；此人不会是知识分子，因为知识分子特有的清高使他不能成为大众的一员；那些凡是不能融入大众、不能成为大众一员的人，就不能对大众生活有所真正体会，也就不能了解大众内心最深层的欲望，更谈不上驾驭大众。”——纳粹党副元首鲁道夫赫斯毕业论文《什么样的领袖能够征服德国？》

在1917年，列宁理解政治的方式是长期生活在忠实的基层布尔什维克党员中间，直接听取他们的声音，与住在豪华洋楼里的布尔什维克其他领导人诸如加米涅夫、季诺维也夫等相比，他更能够挣脱报纸塑造的政治氛围，直接把握普通民众的心态和政治诉求（这是现代政治家最为难得的素质，报纸只是宣传手段，而非对政治局势的客观判断），也由此提出了契合大众心态的政治主张，并获得了基层党员的狂热拥护。

从1917年4月到11月初，列宁多次通过在报纸上发表致布尔什维克基层党员的公开信，来制服党内高层的反对者——在9月底，列宁发布《告工人、农民、士兵书》争取基层党员，支持自己主张，并施压布尔什维克高层。在这个过程中，布尔什维克彼得堡委员会的基层党员，始终都是列宁主张的坚定支持者。

在竞选的过程中，特朗普多次通过发布推特向基层支持者喊话，争取其在党内斗争中支持自己

特朗普则热衷于通过连续举办海量的群众集会，直接接触普通民众，体察他们的想法，然后提出契合党内基层想法的政治主张，并获得了党内基层的坚决支持。

在竞选期间，尽管媒体狂泼冷水，但特朗普多次表示，他集会上民众的热情，表明他获得了相当数量选民的认可

3、他们的政治主张都曾受到党内高层的强烈抵制，但都通过争取基层党员的支持战而胜之，并将自己的政治意志强加于整个党。

当列宁4月回国时，他宣称要夺取俄国政权时（当时布尔什维克实力尚弱），激起了布尔什维克高层的强烈反对：丹尼尔斯在《革命的良心》中这样评价当时的列宁，布尔什维克的领导层将列宁视为“长期脱离俄国生活的空想家和麻烦制造者”；高层干部谢列布里亚科夫最初谈到列宁的夺权构想时，这样评价道，“要么我们是傻瓜，要么伊利奇（列宁）是傻瓜”；孟什维克观察家苏汉诺夫刚开始也认为，列宁当时在党内高层中已经被“完全孤立”（出自苏汉诺夫的《革命札记》）。但是，列宁凭借基层党员的支持，成功扭转了党内的共识，最终使自己的主张成为党的共识。

特朗普在党内的遭遇与列宁同样类似。他激烈的反移民和贸易保护主张，最初在党内高层激起了强烈质疑，但是凭借基层党代表的支持，他最终逼退了反对者，并将他的政治意志强加于共和党。

4、他们都提出主流舆论眼里惊世骇俗、甚至是胡言乱语的政治主张，并不断被国内主流政坛和媒体所指责嘲讽，但他们始终完全无视这些声音。

1917年4月期间，俄国的主要左翼政治领袖克伦斯基、普列汉诺夫、策烈铁里和唐恩等人，认为主张夺权的列宁不过是一个典型的政治白痴。当时克伦斯基甚至懒得拘捕满口叛乱言辞的列宁

2016大选期间，特朗普被美国主流政坛认为是一个傻瓜，图为著名政治脱口秀主持人司徒囧取笑竞选期间的特朗普

列宁在1917年4月初提出了推翻临时政府；10月提出“放弃选票，拿起步枪”；1917年12月坚持对德屈辱议和；1918年1月直接下令解散立宪会议，这些措施都引发了俄国政坛的猛烈抨击。在4月初，圣彼得堡的媒体集中报道了列宁的德国间谍背景，把主张对德议和的他描述成一位面目可憎的俄国叛徒，但列宁却毫不在意（相形之下，孟什维克领袖马尔托夫则因为自己的德国背景患得患失），他轻蔑地回应：“克伦斯基们只知道对士兵奢谈爱国，却不了解士兵们最迫切的需求”，所以，尽管士兵没有能力发出自己的声音，但“他们终将汇聚成革命的洪流”。在10月、12月和18年1月的关键时刻，列宁都展示出对主流政坛和媒体的鄙夷。

在2016年的总统大选中，特朗普同样因为修墙、反穆斯林和贸易保护等政策备受美国主流政坛和媒体的猛烈抨击，但他始终展现出不屑一顾的政治姿态，因为他知道他所要争取的是民众的支持，而非媒体。

5、他们在攀登胜利高峰时，都做出了相当准确的政治规划，并都通过借壳上市的政治手段取得了成功。

列宁通过选举蚕食左翼政党的基本盘，取得了苏维埃的控制权，进而借苏维埃的名义推翻了临时政府，建立了布尔什维克政权；特朗普则通过争取共和党基层选民的支持，跃升为共和党总统候选人，最终成功竞选了总统。

6、他们针对民众的演讲语言，都非常简单易懂。

从当时与列宁演讲风格极为类似的托洛茨基在涅瓦河右岸人民宫的演讲，可以一窥当时列宁演说的风格，托洛茨基说道：

“苏维埃将拿出国家拥有的一切给贫农和战壕里的士兵。资产者有两顶帽子，那就要拿出一顶给战壕里挨冻的士兵。你有暖和的靴子？那就在家呆着吧，工人兄弟需要你的靴子……”

对此，孟什维克革命家苏汉洛夫曾这样评价道，尽管拥有丰富的学识，但列宁和托洛茨基从来不屑展现自己知识分子的优雅，他们完全以“劳动的语言”与民众沟通，所用的言辞不会超越民众认知的范畴。

特朗普的语言风格同样如此，他最常用的单词，诸如“big”、“great”、“pefect”等词，都是最简单的英文单词，是普通民众不需要任何思考就可以直接吸纳的“语言”。

7、他们都经历过灾难性的失败，但都在自身幸运和对手愚蠢的帮助下，卷土重来。

列宁曾经在七月暴动中遭遇灾难性的失败，并不得不潜逃芬兰，惶惶不可终日，一度对革命前景悲观失望，但一个月后的科尔尼洛夫叛乱，则让布尔什维克卷土重来，而临时政府10月中旬试图将首都彼得格勒卫戍部队调往北方战线的命令，直接驱使恐惧上前线的卫戍士兵们向布尔什维克进一步倒戈。

据布尔什维克领导人季诺维也夫回忆，七月暴乱失败后，他与列宁躲在芬兰拉兹利夫湖边，列宁曾自己叹道“现在好了，我们唯一能做的就是装死。”

在2016年10月8日，特朗普猥亵女性的录音被媒体曝光，一时之间，特朗普的支持率急速坠落，特朗普阵营一度濒临崩溃。但到了10月底，奥巴马医保迅速涨价，驱使恐惧涨价的民众选择了支持特朗普，而紧接其后的希拉里亲信泄密导致FBI重启调查，更使得特朗普阵营重整旗鼓，并惊险地赢得了选举。

8、他们都具有强悍的政治意志，雄心勃勃，精力无穷。

在1917年，列宁先后遭到了敌人、盟友和党内亲信的不断背叛和质疑，即便是在不被理解的境地下，他依然坚持自己的政治主张，虽然在具体纲领上做出了一些妥协，但最终目标从未变过。可以毫不夸张地说，在一百年前，是列宁的意志和过人精力，扭转了世界历史的轨迹。

在2016年，特朗普同样遭到敌人、党内盟友和亲信的不断背弃和质疑，在录音门爆出的时刻，他甚至处于众叛亲离的境地，但是，凭借自身的顽强意志，特朗普坚持了下来，在最后选举的时刻，他在关键州坚持每天三场到五场的超负荷群众拉力（希拉里数天内都不能保证一场拉力），并最终赢得了胜利。

9、他们都代表了一种当时的边缘意识形态，这种意识形态的崛起都离不开当时执政者的巨大错误。

在第一次世界大战之前，民族主义在世界各国大行其道，布尔什维克所代表的左翼激进意识形态被严重打压，列宁自己都说，“这辈子不可能看到革命的成功。”但是，战争不但摧毁了执政者的社会基础，制造了动乱，也毁灭了西方民族主义的天然正确性，并最终酝酿了左翼意识形态的崛起，列宁布尔什维克的成功实际上恰恰得益于欧洲大国领导人民族主义过度好战的巨大错误。而第二次世界大战中的纳粹罪行与毁灭，则彻底打垮了白人民族主义的道德基础。

同样，特朗普的登台，也得益于奥巴马、默克尔等当时执政者过度开放的非法移民政策和难民政策。如果没有足以威胁到每一个普通民众日常生活的难民政策负面后果，特朗普就不可能轻易重创左翼思潮的道义合法性，并缔造自己的政治成功。可以预见的是，如同一次大战中持续战争的破坏效应对民族主义的损伤一样，随着难民政策负面效应的不断凸显，左翼意识形态将很难避免走向衰退的命运。

澳大利亚极右翼政客宝琳·韩森（Pauline Hanson）的单一民族党，此前一度在澳大利亚销声匿迹，但随着默克尔难民政策后果的加剧，这一政党开始死灰复燃

作为深切理解现代政治的政客，列宁与特朗普都以群众政治的手段缔造了自己的成功，并开创了属于自己的时代，验证了群众运动所蕴含的无敌力量。1917年10月，伟大的列宁缔造出了基于布尔什维克意识形态的全新国家，由此掀开了世界全面左转的序幕。一百年后，美国新总统特朗普则将他颇具白人民族主义色彩的政治理念，贯彻到一个左翼思潮即将谢幕的旧世界。

事实上，特朗普的上台，并不是世界的福音，甚至也很难说是美国的福音，就像列宁的成功也不完全意味着俄国的美好。但是，对于沉酣已久的民族主义来说，特朗普的成功则是人类政治钟摆时隔百年的久违回拨。
