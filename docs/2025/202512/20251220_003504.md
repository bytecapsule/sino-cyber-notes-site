针对近期庞老捐赠画作数十年后重现拍卖行的事件，我做了一些推测，大家看有没有道理：

1. 关于“真伪”的逻辑悖论 以庞老的江湖地位和眼力，捐出的画大概率是真的。如果是赝品，当年南博接收时不可能看不出来。1961年、1964年两次鉴定为假，极有可能是一个“补救式”的结论——因为在捐赠到鉴定的这段窗口期（59年捐画，61年鉴定为假），画已经被掉包了。

2. 消失的路径：不是“偷”而是“借” 这不太可能是小偷所为，因为失窃必报案；也不是文物贩子能染指的，毕竟是登记造册的国有资产。最大的可能性是：在那个特殊的历史时期，画被某位能量巨大的“ld”以研究或欣赏的名义“借”走了，且此后借故未还。面对“权力大于账目”的困境，博物馆无法报案，为了平账，只能制作一张赝品入库，并通过后来的鉴定将其定性为假，从而在账面上“合法”地销账或降级处理。

3. “传家宝”的最终现身 这幅画销声匿迹数十年，说明它一直被藏在极高门槛的私人手中。第一代持有人深知来路不正，必然交代后辈“只能欣赏，不可入市”。但时移世易，后辈可能并不知道背后的隐情，或者面临财务压力，打破了家训。这才导致这件沉寂几十年的国宝，突然出现在数千万级的拍卖场上。

4. 破局的关键：顺藤摸瓜 这涉及重大国有资产流失，查起来并不难，金额巨大，中间环节的人，谁也不会自己认下来。。拍卖行都有详细的委托人记录。只要查清委托人是谁，向上溯源其家族背景或藏品来源，当年的那段“借画往事”恐怕就能水落石出。

另外，值得注意的是，信息不对称不仅在ld和后代上，而且还有博物院的转接上，老一辈知道怎么回事，断然不会在展览的时候嘲讽庞家后人卖画。。而后一代南博的同志，看到2014年就有庞老的画在卖，就阴阳庞家落魄卖画，当年没有把画全部捐给博物馆，要命就要命在这个嘲讽上，激怒了早已退出文物圈的庞家后人，结果法庭对线，南博的年轻同志出具卖画情况，才发现根本不是庞家后人卖的，而是就是当初捐给南博的文物。。
