最近特斯拉公布的年度更新中第一个功能叫 Grok with Navigation Commands (Beta)，也就是用 Grok 添加和编辑导航目的地。这个功能初听其实没啥吸引力。

我有个哥们 Jackson 是特斯拉 Cybertruck 车主，他的车更新以后做了下面这个短视频，我看他微博没发，要来原素材发一发，大家感受下。

虽然特斯拉带上了（Beta）字符来控制预期，但能看出来 xAI 和特斯拉把 Grok 基于地图、定位的模糊语义理解集成得是比较深的。

看一些 xAI 员工的转发，特斯拉 OS 里调用的模型是 Grok 4.1- Fast，那说明模型这块没有太高的门槛。

技术酷炫是一方面，回到自己的用车场景，我认为也确实有一些场景是用得上的，比如「Grok 帮我搜回家路上的理想或极氪超充站，找最近的那个添加途径点，别绕远了」。

这比Jackson 视频里的需求简单多了，对 Grok 来说是小菜一碟，但目前的语音助手基本是搞不定的，因为涉及模糊语义和多步推理。

而且很容易推演，除了导航，接下来特斯拉完全可以基于 Grok 在车机场景下做一定的后训练，然后让模型执行一系列复杂的任务，从根本上重塑「苹果快捷指令」这样的产品。

让用户直接和语音助手用自然语言说事儿，然后记下来，就和专车司机一样。不要让用户去基于 if-else 去先写一个场景，再写一个执行。 http://t.cn/AXUi8zKs
