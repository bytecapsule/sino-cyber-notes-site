Gartner（高德纳） 刚发布了被称为企业 IT 界的“米其林指南”的魔力象限报告——2025年 AI 应用开发平台魔力象限报告。

一大亮点是除了传统的谷歌、微软、亚马逊、IBM 四大巨头外，中国的火山引擎、阿里云和腾讯云也进入了挑战者象限，占据了三席。

这是 Gartner 第一次发布 AI 应用开发平台这个领域的魔力象限报告，说明 AI 应用开发平台市场已经足够成熟。报告估计 2025 年市场规模超 52 亿美元，年增长率超 30%。

> 注：AI 应用开发平台，简单理解就是 AI 云服务：不用自己买显卡、不用自己训练模型，直接通过 API 调用各种大模型、图像模型和安全护栏。

报告揭示的核心趋势是：AI 应用开发正从实验走向规模化生产。企业不再满足于聊天机器人，而是要构建能处理复杂业务流程的 AI 智能体。

这种报告通常不太容易看懂，但实际上如果你搞明白它的坐标和象限分别代表什么意思就很容易理解了。（参考图2）

魔力象限就跟数学里面的几何坐标一样，两个坐标四个象限。那这份报告来说：

两个坐标轴：

- 横轴"愿景完整性"：知不知道市场往哪走

- 纵轴"执行能力"：能不能把愿景变成现实

四个象限：

- 领导者（右上，既能做又能想）

- 挑战者（左上，能做但愿景欠缺）

- 有远见者（右下，想得好但执行不足）

- 利基玩家（左下，专注细分领域）。

领导者：四巨头各有侧重

1. Google Vertex AI：推出 A2A 协议（已捐给 Linux 基金会），让不同平台的 AI 智能体能互相通信，打破厂商锁定

2. AWS Bedrock：擅长垂直行业方案，医疗、电信、金融都有专门支持，策略是"不做最炫，做最实用"

3. 微软 Azure AI Foundry：生态整合能力强，2025 年增加了 MCP 支持和 Agent Factory 框架

4. IBM watsonx：Foundry Ontology 技术让它在 RAG 和安全性方面领先，走企业级稳重路线

挑战者：中国厂商占三席

这个象限最值得关注——五家里有三家是中国厂商。

1. 火山引擎执行能力排第一。中国公有云大模型调用量占 46.4%，豆包日均 tokens 调用量 5 个月从 4 万亿涨到 16.4 万亿。DeepSeek 火爆时快速支持全尺寸模型，响应灵活。问题是全球品牌知名度不够。

2. 阿里云财务实力强（母公司营收超 1300 亿美元），Model Studio 工具链完整，销售周期平均两周。问题是产品差异化不足。

3. 腾讯云合规认证突出，拥有 SOC、NIST、HIPAA、PCI 等关键认证，适合强监管行业。问题是商业模式迭代慢。

从"挑战者"升"领导者"，关键不是技术（中国厂商已经有了），而是市场战略前瞻性和产品差异化。

值得一提的是，Anthropic 没入选是因为它定位是模型提供商，还不是完整开发平台。类似的，OpenAI虽然入选了，也只是被归类为"有远见者"(Visionaries)，也是因为它虽然在模型能力上很强，但在企业级功能和服务上还有提升空间。

三大关键趋势

协议标准化：MCP 和 A2A 协议的出现是 2025 年最重要变化。以前 Google 平台的智能体和 AWS 的智能体很难协作，现在有了通用语言，打破厂商锁定。

开放权重模型兴起但有悖论：DeepSeek、通义千问、Meta 都在发开放模型，开源框架也很火。但 Gartner 调研显示，73% 受访者用商业平台，只有 25% 用纯代码框架。开源适合原型，生产环境还是选商业平台。

Agent Experience (AX) 兴起：未来应用的"用户"不一定是人，而是另一个 AI 智能体。它不需要漂亮按钮和动画，需要标准化 API 和高效协议。UX 设计将转向同时服务人类和智能体的混合模式。

AI 应用开发平台像当年云计算兴起——开发者不再自己买 GPU 搭基础设施，而是选择成熟平台。魔力象限不是标准答案，但是张好地图，帮你看清格局和方向。

国产平台已在全球市场占据重要位置，技术上不输人。但在全球化运营、生态建设、产品差异化上还有空间。特别是标准制定——MCP 是 Anthropic 的，A2A 是 Google 的，希望未来有更多中国厂商参与 AI 国际标准制定。
