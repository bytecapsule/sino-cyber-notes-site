为何中国人的 Openclaw 的八个牛马在 discord 里，而不是微信和QQ？

1

随着 2023 年 GPT 的横空出世以及最终 Openclaw 这种可用的 AI Agent 最终落地，Discord 这个 IM 客户端越来越展现出了非凡的远见。

他对 bot 的良好的支持和用 hashtag 灵活的组织话题的能力，让我的一个朋友可以轻松的在里面操作八个 Openclaw 牛马干活。。。（图1）

2

而和你想象的不同，这不是 discord 拥有乔布斯这种穿越者，而是因为 Discord 拥有一种“兴趣社区广场”的文化基因。

@\#Rapptz（非微博用户） 早在 2015 年就逆向了 discord 并且发布了 discord.py 机器人库。

而 Discord 官方的回应不是律师函，而是一篇奠定了 AI 时代 Discord 地位的文章：

《机器人革命已经非正式开始了》。

The Robot Revolution has Unofficially Begun(Unofficial API)

在这里 Discord 官方为用户介绍这些 bot 并且承诺官方 API 正在研发中。

3

自此，蓬勃的 discord bot 社区飞速发展。

中间出现过各种机器人攻击，最终限制 bot 速率，rapptz 因为官方开发的一些要求曾经放弃 discord py 项目等等事情。

但是同时，官方在积极的学习这些野生项目并且完善自己的功能。

实际上，很多人担心的什么攻击和隐私问题，从 discord 2016 年的官方 Bot API 和 OAuth2 开始，都已经逐步解决了。

4

2025 年底，Openclaw 横空出世。

所有人突然发现，discord 好像就是为了 AI Agent 的到来而生的。

虽然， 2015 年没有人能遇见我们 2023 年会搞出涌现了能力的 LLM ， 2025 年也会搞出可用的 AI Agent。

但是，人家 2015 年的文章的标题，说的没错啊：

机器人革命，已经非正式开始。

5

这里，就必须有请知名的法律公司，哦，不对是科技公司的产品微信了。。。

经过这些年不懈的努力，蓬勃的微信和 QQ 的 bot 开发社区生态。。。

几乎已经团灭。[666]

当然这里有灰产，但是就像 discord 已经证明的一样：

很多问题，其实是用技术手段和开放平台就能解决。

2020 年知名QQ机器人“晨风”的作者被跨省，含泪求助法律支援其实就在 AI 技术革命的黎明之前。

而类似的还有 WeTool、联络易、微商助手，晨风机器人、酷Q 等等等等，本来微信和 QQ 是有一个绝对不会比 discard 差的机器人生态的。

无奈，机器人革命即没有非正式开始。。。

而很多诉讼，已经正式开始。。。
