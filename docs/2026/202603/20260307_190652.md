研究openclaw，有一个好处，就是你再看别的智能体，会有一种熟悉的感觉。

因为大部分智能体，目前都是那些原理，可能具体实现的手段有所不同。

分享一个写网文的智能体系统：Webnovel Writer。

Webnovel Writer 是基于 Claude Code 的长篇网文创作系统，目标是降低 AI 写作中的“遗忘”和“幻觉”，支持长周期连载创作。

你看它的文档，你就会觉得非常熟悉。

详细文档已拆分到 docs/：

架构与模块：docs/architecture.md

命令详解：docs/commands.md

RAG 与配置：docs/rag-and-config.md

题材模板：docs/genres.md

运维与恢复：docs/operations.md

文档导航：docs/README.md

龙虾也有各种人格、记忆模版，因为它是个人助理，当然需要这些模版。

但是一个写网文的智能体，就不需要这些。

需要的就是大纲、题材等等模版，功能方面也更需要RAG来获得各种参考资料。
