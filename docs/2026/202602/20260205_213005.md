为何 OpenClaw + skills 会席卷很多行业？

因为 LLM 虽然在训练的时候几乎使用了人类在互联网上的所有数据，但是，LLM 有一个重要的数据缺失，就是真正专业领域的人类 know how 。

而这些 know how 是不会有人贡献出来给 Open AI 这种私有企业的。这在经济上根本不合理。

但是，skills 允许私人 know how 和模型的基本智能结合起来，单独为了自己的业务混合运算。。。

所以无论你是 IT 资深架构师，资深产品经理，还是资深投资专家，你都可以用一个简单的 markdown 文件描述你的 know how ，并且指导模型运用自己的基本能力和通识。

人类更核心，商业价值更高的非公开的行业 know how，就被利益，给全部整合为了目前在迅速膨胀的 skills 。。。

这件事的意义不仅仅在于会颠覆 LLM 的能力边界，会颠覆什么 SaaS 行业，这件事的意义更大。。。

他可能会彻底颠覆 Open Source 这种软件时代全球协作的范式。

也会终结过去几十年只有通过专业 IT 人士的“翻译”，才能大规模运用算力的限制。
