`AI前沿速递`  一款将网页变成干净 Markdown 的免费神器：markdown.new。可以将任意公开网页一键转换为干净、结构清晰的 Markdown。它会自动去除广告、脚本和多余样式，保留标题、列表、链接、表格等核心内容，让网页更加轻量，体积最高可减少 80%。无论是阅读、整理资料，还是后续加工处理，都更加高效。在使用 AI 对网页进行翻译、总结或分析时，也能大大降低 Token 消耗。

使用方式非常简单，无需注册

1. 浏览器使用：在任意网址前加上 markdown.new/ 即可。

2. API / cURL 调用：发送一个 JSON 请求，例如 {"url": "https : //example.com"}。

可通过 method（auto、ai、browser）选择转换方式，也可以用 retain_images=true 控制是否保留图片。

这款工具长期免费，无需订阅。每个 IP 每天可请求 500 次，超出后会返回 HTTP 429。可以通过 x-rate-limit-remaining 查看剩余额度。如有更高的调用需求，也可以自行部署它的开源版本。

在技术实现上，系统采用三层自动兜底机制：优先请求网页原生 Markdown。如果返回的是 HTML，则进行智能转换。遇到依赖大量 JavaScript 的页面，则通过浏览器渲染获取完整内容。整个过程会自动选择最快且可行的方式，确保稳定高效。

需要注意的是，目前仅支持公开页面，付费或需要登录的内容无法转换。图片默认不保留（可手动开启）。对于体积过大的页面可能会出现截断，部分重度依赖 JavaScript 的网站也可能增加约 1–2 秒的处理时间。（零重力瓦力）http://t.cn/AXcG75zE
