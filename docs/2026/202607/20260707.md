《经济学人》：女性在职场上的进步似乎正在停滞——

文章称，过去几十年，发达国家女性取得了令人瞩目的进步。

如今，在几乎所有富裕国家，大学校园里的女生人数都已超过男生。

在职场上，许多女性也积极践行Sheryl Sandberg2013年畅销书《Lean In》中倡导的理念——勇于争取机会、积极竞争领导岗位。

自1980年以来，美国女性在医生、律师等高薪职业中的占比几乎增长了三倍。

在英国，世纪之交时，病人遇到男性医生的概率还是女性医生的两倍。而根据英国General Medical Council的数据，去年英国女性执业医生人数首次超过男性。

然而，近几年，人类历史上受教育程度最高的一代女性，却越来越多地表现出一种“向后退（lean out）”的趋势。

2024年，S&P Global的一项研究发现，美国上市公司女性高管占比在2023年出现近20年来首次下降。

2025年，美国S&P 500企业新增董事席位中，女性占38%，低于上一年的42%，并延续了自2020年高点以来的下滑趋势。

华尔街女性晋升开始放缓

在2024年最新一轮合伙人晋升中，Goldman Sachs晋升的女性合伙人比例，十多年来首次低于上一轮。

一年后，这家华尔街投行新晋女性董事总经理的比例也再次下降。

上个月，被视为最有希望接替Jamie Dimon执掌JPMorgan Chase的女性高管Marianne Lake职位被调整；另一位热门候选人Jennifer Piepszak则于去年退出竞争。

结果，接班人名单变得异常“白人且男性化”。

不只是高层，普通女性也开始退出职场

文章认为，女性职业发展的停滞甚至倒退，并不仅仅发生在管理层。

根据Organisation for Economic Co-operation and Development（OECD）33个成员国的数据：女性全职就业率从2023年的78.1%下降至2024年的76.8%。虽然降幅不大，但这是PwC15年前开始统计以来首次出现下降。

美国方面：2026年3月，25—54岁女性劳动参与率达到历史最高的78.5%。但此后几个月下降了接近1个百分点，降幅明显大于男性。

高学历、有幼儿的母亲退出最明显

美国拥有大学学历、育有年幼子女的女性，似乎正成为这一趋势的主力。她们几十年来持续上升的劳动参与率在2023年出现拐点，此后连续下降。

去年也是过去40年来幼儿母亲劳动参与率下降最明显的一年。

男女薪酬差距再次扩大

近年来持续缩小的男女收入差距，也重新开始扩大。

美国：2023年扩大；2024年再次扩大。

这是60年来首次连续两年恶化。

2024年，加拿大、法国、瑞士等多个发达国家也出现类似情况。

英国方面，那些40多岁、当年正是在桑德伯格鼓励下"向前一步"的一代职业女性，其薪酬差距也再次扩大。

女性的职业抱负开始下降

女性的职业目标如今似乎也开始落后于男性。

McKinsey & Company与桑德伯格创立的非营利组织Lean In于2025年底发布调查显示：

2019年至2023年：无论男女，希望获得晋升的人数比例，都从约70%上升到80%。

但到2025年：男性接近90%仍希望晋升；女性却没有继续增长。

基层女性员工中，仅有69%表示希望升职。

普华永道的调查也发现：女性主动争取更高职位的意愿持续低于男性。

为什么会出现倒退？

文章认为，原因可能有几个。

1、疫情造成统计效应

劳动经济学家指出：新冠疫情期间，大量低薪女性失业。这在统计上人为缩小了男女工资差距。疫情结束后，这些女性重新进入劳动力市场，因此工资差距有所扩大。

不过，疫情结束已过去数年，薪酬差距仍继续扩大，说明还有其他原因。

2、托育资源不足

另一个重要原因是：幼儿照护越来越难获得。

虽然一些发达国家通过补贴降低了托儿费用，但托育机构供给仍然不足。结果就是：排队等待时间长；护理人员短缺。

例如德国虽然一直被认为拥有较完善的托育体系，但目前仍缺少30多万个三岁以下儿童托位。

3、DEI政策退潮

与此同时，尤其是在特朗普执政下的美国，“多元、公平与包容（DEI）”逐渐成为一个带有政治争议的词汇。联邦政府正公开拆除DEI政策。

许多企业也在更加低调地减少相关措施，以免与这位强调传统男性气质的总统发生冲突。

《经济学人》评论道：几年前，作为进步主义商业领袖代表之一的杰米·戴蒙，大概很难想象自己公司的CEO接班人名单会清一色全是男性。

文章认为，过去几十年女性教育水平和职业机会不断提高，但近两三年这种进步出现了明显放缓甚至部分逆转。

具体表现包括：女性高管比例停止增长；女性劳动参与率下降；男女薪酬差距重新扩大；女性主动追求晋升的意愿减弱。

《经济学人》认为，这一变化可能是疫情后遗效应、托育资源不足以及美国企业DEI政策退潮等多种因素共同作用的结果，而不仅仅是单一原因。

Anthropic Claude Code 工程师 Thariq Shihipar 上周在 AI Engineer World's Fair 上做了一场关于 Fable 5 的演讲。

他提了四个主题，最有信息量的是前两个。

第一个叫“解除 Claude 的束缚”（unhobbling Claude）。

他的核心观点是，模型的能力是在使用过程中逐步摸索出来的，很难一开始就设计出来的，所以模型变强的方式往往出人意料。

举例来说：你问聊天模型哪些宝可梦的名字以 aw 结尾，它答不上来，因为它虽然知道所有宝可梦的名字，但没法在脑子里一个一个过。但如果你给它代码执行工具，它会去拉全部宝可梦列表然后写脚本过滤，两秒钟就能找出答案。

Anthropic 内部把这叫“能力悬余”（capability overhang），模型其实已经能做很多事，只是我们还没找到正确的打开方式。

Claude Code 最近一个关键变化是砍掉了 80% 的系统提示词。早期模型需要详细的指令和大量示例，但 Fable 这个级别的模型反过来了，给太多示例反而会限制它，因为它自己的想象力比你给的示例更丰富。新的做法是给上下文，不给约束；告诉它情况，不告诉它不许做什么。

第二个主题叫“找到你的未知”。

你写给模型的提示词就像是“地图”，但真正的代码库和现实世界是“领地”。模型在领地里碰到地图上没标注的东西，就是一个未知数，就得自己做决定。Fable 5 的活动范围太大了，如果你不提前搞清楚这些未知数，它会在你没想到的地方做出你不想要的决策。

他给了几个具体方法：

让 Fable 做一次“盲区扫描”（blind spot pass），在动手之前先通读相关代码，帮你找出你自己都不知道的潜在问题。

让它一口气做四个风格完全不同的原型，这样你可以通过反应来发现自己的偏好，而不是先想清楚再描述。

让它提问你，通过提问把你脑子里那些"知道但没写下来"的细节挖出来。

给它一段别的系统里的代码当参考地图，比直接写规格说明书更高效。

让它在执行过程中记录每个偏离你预期的决策点，事后你能看到它在哪些地方遇到了你没考虑到的问题。

最后让它反过来考你，确保你理解它做了什么，这样你在提 PR 的时候能说清楚发生了什么。

演讲的后半段内容就偏感性了，有点像@onevcat 写的那篇《当编程变得不再有趣》 网页链接。

他说第一次用 Fable 的时候，既觉得获得了很多，也觉得失去了什么。他以前开过一家三十人的 YC 创业公司，当时团队因为写代码太难，被迫在各种功能之间做取舍。几周前他回去看那个代码库，用 Fable 几小时就做完了当年要花好几周的事。他真的很喜欢手写代码时那种在脑子里旋转整个代码库的感觉，但也记得那些熬夜调 bug、项目一个接一个失败的日子。

他最后的建议是“打破常规”（be unreasonable）。

以前做取舍是本能，好、快、省三选二，现在他觉得这套逻辑该被推翻了：与其先设好优先级排除掉一部分，不如逼一下现实，看它是不是真的会逼你二选一。他演讲前一天花四个小时用 Fable 做完了这场演讲的全部 PPT。 宝玉xp的微博视频

\#今天要来点物理吗？\# 

14年前的昨天，欧洲核子研究中心宣布发现了希格斯玻色子。

有人问：这又如何以任何方式影响了世界——它有什么意义？

希格斯玻色子的深远影响没有得到更好的传达。

它之所以深刻，不是因为它出人意料，而是因为它提供了压倒性的证据，证明我们宇宙的真空在深层意义上表现得像一个超导体。

在超导体中，库珀对形成的序参量破坏了局域 U(1) 对称性，导致原本无质量的电磁模式在材料内部表现为“有质量”（即麦斯納效应，电磁场被排斥）。物理学家把这与希格斯机制类比：局域规范对称性被破坏后，规范玻色子获得有效质量。这个思想被称为 Anderson–Higgs 机制。 

把“质量的来源”从形而上问题变为可检验的物理机制，把粒子物理的标准模型补全为一个自洽的理论框架；同时把高能物理与凝聚态物理（超导）联系起来，显示不同尺度下相同的对称性破缺原理。 

把真空想成“有粘性的介质”（希格斯场），粒子在其中运动会“被拖慢”——表现为获得质量；超导体里光子在材料中也“被赋予质量”，两者数学结构相似。
