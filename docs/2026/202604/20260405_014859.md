\#今天要来点化学吗？\# 

30年前在村上春树的一篇随笔里读到，美国人喜欢在可口可乐里加花生。

这种喝法在美国南部有一定传统，通常是把咸花生直接倒进玻璃瓶装的可乐里。这被称为“农夫可乐”。

在文化记忆里，它常被视为一种怀旧的“南方小吃搭配”。不过现在大多数美国南方人甚至也不了解这些文化。

不过美国农夫认为这种搭配很美味，并非是他们的错觉。

经典版可口可乐的pH值为2.5，与胃酸的酸度大致相同。当把烤花生放入可口可乐中时，其中的磷酸会使花生表面的蛋白质部分变性，释放出游离的谷氨酸。这样，鲜味就在杯中实时产生。

花生上的盐分会抑制舌头上的苦味受体，从而增强你对甜味的感知，而无需额外添加糖。可口可乐每罐已含有39克糖。由于盐分清除了其他味觉信号的干扰，你的大脑会感觉它更甜。所以要选咸花生。

碳酸化作用有两个方面。首先，溶解在液体中的二氧化碳会形成碳酸，碳酸会刺激痛觉受体（TRPA1），而不是味觉受体。这种轻微的刺激感能让你的味蕾在每次啜饮之间得到休息，从而避免味觉疲劳。每一口都像第一口一样令人惊艳。其次，气泡会搅动花生表面，加速蛋白质分解和谷氨酸释放。花生浸泡的时间越长，提取出的鲜味就越多。

脂肪含量是关键。花生中脂肪含量高达49%（按重量计）。脂肪是唯一能激活CD36受体的宏量营养素，而大脑会将这种受体解读为浓郁和满足感。将脂肪与糖、盐、酸、鲜味和碳酸混合在一起，就意外地同时激活了人类味觉系统中所有主要的奖赏通路。

20世纪20年代，佐治亚州的农民把花生米丢可乐里是为了空出一只手。他们偶然发现了盐-酸-鲜味-脂肪-碳酸的最佳循环，而食品科学却在一个世纪后才能够解释其原理。
