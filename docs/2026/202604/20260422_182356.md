腾讯开源了一个给AI Agent用的沙箱服务：CubeSandbox
地址：github.com/TencentCloud/CubeSandbox/
Cube Sandbox 是一款基于 RustVMM 与 KVM 构建的高性能、开箱即用的安全沙箱服务。它既支持单机部署，也能够很方便的扩展到多台机器的集群服务。
特点是性能和安全：冷启动一个沙箱时间平均 < 60ms、单实例内存开销低至 <5MB，可在一台机器上跑起数千个 Agent。每个 Agent 拥有独立的 Guest OS 内核。
