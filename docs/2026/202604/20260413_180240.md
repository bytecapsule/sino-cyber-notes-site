总是有一群脑残反对中英夹杂，但是今天 x 上开始有人讨论 prompt 如果中英夹杂的效果会更好。

对此我一点都不惊讶，因为我给 Gemini Pro 的脑暴阶段的 prompt 就是中英夹杂的，效果会明显不同。

而且这个不是民科经验，从模型原理上很好解释，中英夹杂有机会激活更多神经网络。激活节点数和激活模式多样性是不同的（具体效果可以参考 refs 给出的 paper）。

所以，如果你需要模型做类似 brainstorm 这种创意性工作，那么不要改变你的中英夹杂的语言习惯。

至于那些评论你的 sb，没必要在意，ai 时代，他们很快 out 。
