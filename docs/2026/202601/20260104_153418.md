面对 VS Code 日益膨胀的体积和内存占用，有时只想快速修改几个文件，电脑风扇都跟着狂转。

偶然在 GitHub 看到 Horizon 这个开源项目，基于 Tauri 2.0 和 Rust 构建的代码编辑器。

主打极速启动与低资源占用，内置代码高亮、文件搜索、目录导航等基础功能，还集成了终端支持多实例运行，可以边写代码边执行命令。

GitHub：github.com/66HEX/horizon

文件管理方面，支持树状导航、多标签页切换，还能预览图片和播放音频文件。终端功能也很完善，支持命令历史记录、URL 检测、ANSI 转义序列处理。

目前工具处于早期开发阶段，功能还在完善，支持 Windows、macOS 和 Linux 系统。

对于厌倦了臃肿 IDE、追求极致启动速度的朋友，可以尝试一下。
