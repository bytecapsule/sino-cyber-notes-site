AI领域的大事件。

Agent Skills 成为开放标准，OpenAI、微软、Google 和 AWS 全加入了。

经过漫长的实践，人们逐渐总结出一套使用AI的标准化方法，完全可以用于生产领域。

Agent + Skills + MCP

第一层：连接层（MCP）

AI 通过 MCP，可以连接各类外部工具和数据，CRM、网盘、数据库、财务报表、搜索....

第二层：知识层（Skills）

Skills 是一套非常专业细致的AI提示词，能处理各类复杂问题。

第三层：编排层（AgentKit/Agent Builder）

可视化工作流设计，多Agent协调，负责全局管理。

———

举个生活化的例子，管家相当于 Agent ，手下有清洁工、厨师、保安... 这些专门干特定工作的就是 Skills，遵照详细的工作手册进行干活。一些工作需要和外边人打交道，这就是MCP。
