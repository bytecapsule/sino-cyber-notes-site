这与 Claude Code 有什么不同？

虽然在功能上与 Claude Code 非常相似，但有以下几个关键区别：

1. 100% 开源

完全开源，无需依赖任何特定服务提供商。

2. 不绑定特定提供商

虽然推荐使用 Anthropic，但 opencode 同样可以与 OpenAI、Google，甚至本地模型配合使用。随着模型的发展，它们之间的差距会逐渐缩小、价格也会下降，因此保持对提供商的中立性非常重要。

3. 专注于 TUI（终端用户界面）

opencode 由 neovim 用户和 terminal.shop 的创建者开发，我们会充分挖掘终端界面的潜力。

4. 客户端/服务器架构

例如，你可以在电脑上运行 opencode，同时通过移动应用远程控制。这意味着 TUI 前端只是众多客户端中的一种。
