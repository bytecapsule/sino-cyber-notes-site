用另外一种办法来描述马克思的劳动价值论的正确性    [数学]  2006-07-04 

我以前用了一个理发店来描述马克思的劳动价值论的正确性，就是商品的价值就是生产出这个商品的社会必要劳动时间。只要我用一种办法让大家相信这一点了，马克思的劳动价值论就可以被认为是正确的了。 

假设我们在社会上找到任何一家运行稳定的企业，或者公司，或者个人，都可以。不妨假设就是公司吧。所谓的运行稳定，我是指的零增长，就是说，一个企业的规模即没有扩大，也没有缩小，盈利情况也是稳定的，年复一年地那么运行着，即没有变大的趋势，也没有倒闭的趋势。 

这样的企业应当是存在的。而且，可能社会上大多数公司，都具有一定的稳定性。比如可能你家对面有一个饭店，已经开了多年了，规模即没有变大，也没有变小，总是那个规模，那个商品，那种服务。 

所有的企业一定是要买进一些东西（当然，也可以买进的东西为零，比如，假设按摩吧，但我们不妨假设总是要买一些东西的，即使是按摩也要有按摩床），然后经过加工（或者零加工），出售一种叫“商品”的东西，这里的“商品”完全可以是无形的，比如说，就是某种服务，如搓澡，或者理发，等等。 

我上面说的“买进东西”，这里的东西也是抽象的，包括登广告，可以认为是“买进广告”，甚至包括“纳税”，也可以认为是“买税”。因此，房租水电装修电话，等等一切，都包括在这个“买进东西”里面。因此，我们不妨把买进东西也可以称之为买进一些商品。虽然纳税好象不是一种商品，也不妨视为一种商品吧。 

我这里暂时没有把人工或者劳力也放在买进的东西里，现在我可以加进来，比如说，一家企业雇了五个人，意思当然也是要签定劳务合同。而劳动力的价格，也是由市场上的行情来确定的。 

马克思的理论将买进的除劳动力外的其它商品称为不变资本，称劳动力的价格为可变资本。 

当然，不同的劳动力，价格是不同的，比如技术人员，可能月薪一万，而日杂工人，可能月薪五百，等等。 

此外，资本家也是要劳动的，因此也有一个劳动力的价格问题。资本家做的是管理的工作，因此，劳动力的价格应当是相对高一些的。可是资本家这个人的劳动力价格，应当是多少呢？可以这样计算，就是说，这个公司稳定运行的时候，这个资本家挣到的钱，全部用在自己的生活上的费用，给出一个数，比如说每月要花二万元生活。如果这个资本家辞了职去给另一家公司作管理工作，月薪如果是三万元，则这个资本家就不愿意开这个店了，愿意去那家公司上班了。当然，如果另一家公司给出的管理工作的薪水是一万元，则这个资本家就会觉得还是自己开店的好。因此，也能够获得一个资本家劳动的劳动力价格。 

我上面已经讲了，这家公司是稳定运行的，在稳定运行的情况下，应当是即不盈利，也不亏损，总是那个样子。我说的不盈利，是指的资本家的劳动力价格没有被视为利润的情况。意思也就是说，这个资本家的在外面的劳动力价格是月薪三万，自己开店每月的收入用在自己的生活费上的也是三万，除此之外没有任何其它的盈利，惨淡经营了。 

那么，这个公司既然不盈利也不亏损，则它出售的商品的价钱，就必然是两部分组成，一部分是劳动力的价钱，包括了资本家的劳动力，另一部分，就是除劳动力之外的购买的所有东西，如原料，房租，水电，缴税，等等等等非劳动力的东西，正如前面说的，第一部分叫可变资本，我们用V表示，单位当然是“元”，第二部分叫不变资本，我们用U表示，单位也是元。因此，这家公司出售的商品的总价格，应当是V＋U。 

如果价格大于这个V＋U，那就是产生了盈利，如果有了盈利，别人肯定也不傻，也就会开同样的店，做同样的事情，一直到价格又降回到V＋U为止。如果有了亏损，则资本家会选择关门大吉而去受聘于其它的公司，当然，也可以以低的收入坚持一段时间，盼望将来能够盈利找补回来。 

因此，在商品价格的两个成份，V和U中间，我们现在要争论的课题是，马克思的价值论有可能是错误的，商品的价值不仅仅包括了劳动力价值，还包括了一些其它的莫名其妙的东西，这是为资本家阶级辩护的人给出的猜想。 

那么，U这一部分，是购买来的东西，是不变资本，比如原料房租水电通信广告等等，这部分，我们暂时假设它们的价值究竟是不是完全由社会必要劳动时间决定的，我们暂时是“我不知道”这样的态度，用一个专门的术语叫“待定”吧。但是可变资本这一部分，就是V这一部分，完全是劳动力价格。因此，这一部分我们完全相信是由马克思的想法，社会必要劳动时间构成的，通常在商业中简称为“人工”嘛。 

因此，如果我们能够说明，社会上的所有的企业，公司，它们生产出来的商品的价格的两个组成部分，U和V，主要是U，都是在实质上是由“人工”，构成的，我们就已经成功地为马克思的劳动价值论辩护成功了。 

可是前面我们已经讲了，所有的公司，买进东西的价格都是U，卖出东西都加进的是V，因此，一系列的接力的生产，在这个过程中加进的统统都是V。 

我们不妨从最初始的公司查起，比如说有一种商品的原料是沙子，而假设那个地方就是大沙漠，沙子是没有人卖的，因为多得是，但是，如果是这样，则沙子的进价可以视为零，而其它的不变资本不过就是生产工具什么的，又是从其它公司购买来的。那么，因为是其它公司购买来的，那些商品又一定是两个部分组成，U＋V。 

因此，当社会上所有的公司的生产出来的商品的价格统统都是U＋V的时候，也就是商品的增值部分统统是由劳动力或者可变资本转化而来的时候，我们别无选择地认为U中间也就不可能有任何别的成份，只能是一系列公司的一系列的可变资本，一系列的V的累加而构成。 

因此，马克思的劳动价值论是正确的。
