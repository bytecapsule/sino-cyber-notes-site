毛泽东时代是这样被歪曲和否定的-读美丽强国网友《个体与集体》心得（一） 

右而左 

标的帖及其位置：个体与集体，兼回右而左网友 (美丽强国: 2006-05-27 20:46:25)6756B (0/128/3)；http://bbs.people.com.cn/bbs/ReadFile?whichfile=1396339&typeid=17 

我知道美丽强国网友是我的一位老读者，但对我的观点通常持有异议。我推测他是一位严肃认真的人，所以，虽然其《个体与集体，兼回右而左网友》一文，令我啼笑皆非，但我还是十分尊重他本人。他的这一贴我已经跟过一个贴，本可以不再理会，但回头再一看，还是觉得有必要写点体会，理由有二：1，其太多不当掩盖在许多生硬堆砌起来的美丽词汇中，还被覆盖了一层马克思和列宁的外衣，云山雾罩，恐怕一些读者还以为有理有据，头头是道；2，他文中表现出的对于两个时代的不少认识，在特定人群中带有普遍性，一旦给与适当的分析，可以帮助其他人理解，这个特定人群到底有多么无知却无畏，毛泽东和他的时代到底是怎样被他们歪曲和否定的。 

先要说明，他的这个帖子是回我的《自我觉悟，还是自我蒙昧？--与美丽强国网友商榷》帖子的。我那个帖子是对美丽强国网友如下观点的反驳： 

毛泽东时代把人变成工具，后一个时代把工具还原为人，工具是不能评价历史的，只有自我觉悟，摆脱蒙昧，成为自我的非工具的人（精英），才能评价历史，才能看到两个时代的差别。 

这个观点是我根据他对我的《刍议“两个时代”之五》跟帖的观点归纳所得。对此他没有提出质疑，说明他同意我的归纳。 

我在对他的观点展开反驳时，没有超出他的核心观点所涉及领域之外，提出新核心观点，只是对他的核心观点从各个侧面进行了否定论证，并在论证中提出一些支持性或辅助性次级观点。他的《个体与集体》本应该是对我的反驳的回驳，如果他不同意我的反驳，就应该站在自己原来的观点的立场上，论证其结论是如何得出的，同时对我进行回驳，但遗憾的是，他没有这样做，似乎迷失了自己的核心观点，却在钟情于对我的支持性次级观点展开猛烈开火时，错误百出，不得要领。他是努力的，但结局令人同情。 

【（右而左网友的主要观点）其一，个人的经验不可信，哪怕是亲身经历的事情。其二，毛时代，特别是文革时代是个启蒙的时代，是个人“自我意识的一次全面唤醒”，即通过所谓“集体意识（表现为国家）是个体自我意识的加速器，个体在这个加速器里被加速，为集体（国家）贡献力量，使自我意识得以申张。”其三，改革开放时代是个人重新被蒙昧的时代，特别是精英人物的自我意识蒙昧。用他的话说，到底哪个时代人变工具，还是工具变人是很明晰的，那就是，当然毛时代是工具变人，改革开放则“把人还原为工具”。】 

看过以上他对我的观点的归纳，后两点果然与他自己的观点正好对立，说明我在“拽”（他对我的批评用词）出的几千字中，始终没有偏离主题，反驳论证是到位的。我是反驳他的观点的，要是与他的观点不对立那就怪哉！所以我的观点不需要归纳。这是我啼笑皆非的原因之一。原因之二，是他的这个“其一”： “个人的经验不可信，哪怕是亲身经历的事情”。我什么时候说过这话？又怎么能从我的话里推导出这话？我说的是：经验重要也不重要，眼见为实也为虚......认识论的方法比个体的眼睛更具客观性。我明确指出了“过来人”经历，包括我自己的经历的有限可靠性（他强调自己是过来人，所以我礼尚往来，告诉他我也是“过来人”，还站在“四个分水岭”上），并以没有目击车祸的警察，对车祸具有最终判决权为例，说明这一点。我跟帖，回应了这个认识论问题，此处不再重复，仅再一次强调，我谈的不是经验本身多大程度上可信不可信，而是在谈认识问题必须要遵循从材料到感性认识再上升到理性认识，完成“两个飞跃”，尤其是后一个“飞跃”，才能获得真知灼见，为我后面的论证提供方法论指导。 

更让我啼笑皆非的是后面许多个逻辑和认识上的重大不当，甚至重大错误。根据这些不当或错误逻辑推导出的结论，可以让我们每一个人瞠目结舌。先看看下面这一段： 

【虽然右而左网友反复不厌其烦地说本人是精英分子云云，但本人从来都不认为自己就一定比如说比右而左网友更高明，但也从来不认为一定就比他更愚昧，如果人人都是平等的，则最后的结果无非就是半斤对八两，按照这个逻辑，就完全说不上谁更正确了。这也就引出了一个判断的标准问题，其实也就是历史评判的主体性问题。是哪个个人还是集体是历史的评判主体，或者说谁有资格对历史做出评判呢？】 

这一段看去是在说，我和他不过半斤八两，我的观点未必就比他正确。但实质上，他是把精英论历史观，即只有少数他这样“觉悟的精英主体”才具有评价历史的资格和能力，包装在平等的美丽词汇之中。请看我的分析。 

【 “如果人人是平等的，则最后的结果无非是半斤对八两，按照这个逻辑，就完全说不上谁正确了”。】先看这句话的逻辑错误。所谓平等，一般而言，是指取得公民资格的社会成员，在法律面前人人平等，如平等的选举权和被选举权，婚姻双方具有同等的婚姻处置权等等，而他这里所谓“半斤对八两”，结合上下文，显然是在说，任何人天赋的和后天努力所得的智力、智慧、认识能力等是绝对一致的，没有高低、多少之分。

大家注意到了没有，用来立论的前提是个法制概念，得出的结论却是一个认识论能力概念。这就好比是说“如果苹果都是绿色的，则桃子必然个个重量一致”。荒唐不荒唐。按照这个荒唐的逻辑推导，我们得到这样的结论：右而左和华罗庚是平等的，所以右而左和华罗庚（在数学问题上）的见解，半斤对八两，因此，让华罗庚去为核武器搞数学计算和让右而左去做同样的计算没有差别（乖乖，我非把中国的核工业毁了不成）。

美丽强国和孔子是平等的，则美丽强国和孔子（哲学思想）就是半斤对八两，因此，今天孔学可以以美丽强国的阐释为终决阐释（美丽强国网友，你能胜任吗？）；毛泽东和蒋介石是平等的，则毛泽东和蒋介石（政治军事天才）是半斤对八两，因此，蒋介石不会败给毛泽东，他败走台湾，是捏造历史事实。（这里“因此”之后是我帮助完成的进一步推理）

按照这个公式，举一反多，不知道会闹出什么样的笑话，读者不妨自己把爱因斯坦、牛顿等杰出人物都按自己的想象力套进去一试。美丽强国网友本来是要借“平等”，借“半斤对八两”，来平衡我的所谓“四个分水岭”的比较优势，却落入了这样一个荒唐局面，没有察觉。既然这一切都不成立，美丽强国网友的逻辑基础坍塌了，他以此为前提，推导出的其他结论就统统不能成立。但是，他却生生堆砌词汇，并借这些堆砌的词汇把自己又抬高到一个比我高的位置，与他自己的所谓的“平等”和“半斤对八两”的前提发生矛盾。 

请注意接下来这句话：【 “这也就引出了一个判断标准问题，其实也就是历史批判的主体性问题。是哪个个人还是集体是历史的评判主体，或者说谁有资格对历史做出评判呢？”】。虽然他用了“这也就引出”这个承接语，这句话与前一句话还是建立不起逻辑关联。

但就其本身而言，显然的事实是，美丽强国把“评价历史的判断标准”等同于“评价历史的主体”。略有常识的人都应该知道，评价历史的判断标准应当有客观的尺度，这个尺度外在于评价历史的主体（认识主体），可以被主体所认识、所掌握、所使用，但它不能等于主体或主体的主观认识。

因为主体是无法摆脱主观局限的，设立客观标准正是要克服主体的主观认识局限性。这里标准是连接客体（认识的对象，即历史）和主体（人）的桥梁。也就是说，足球裁判员，哪怕偏向他正在执裁的某个队，也不能把足球规则（外在的客观标准），扔了不管，把自己的主观执裁意向（美丽强国的所谓主体），设置为执裁标准。道理就这么简单。

但不幸的是，美丽强国网友认为标准和主体一致。我不认为这个知识或逻辑的刚性错误，仅仅来自他的知识不足或逻辑混乱，也不能将其简单归于“唯心主义历史观观”，因为它与唯心主义历史观风马牛不相及，唯一合理的解释是来自他和那个特定人群里的其他人共有的深层心理动机，是他们心理某种潜意识的显意识化。

这个深层心理动机是什么？就是人其实不都是半斤对八两，也不都是平等的。让我分解一下就知道了。他（意识中）先把人分成几个群体，如农民、工人、官员、被毛冷落甚至“迫害”过的知识份子（后来都加入到一般所谓精英知识分子群体）、一般知识分子等等，然后选取被冷落和被“迫害”过、后来被请为座上宾的这个群体（方便起见简称“反毛的精英知识分子”），作为唯一有历史评价资格和能力的所谓主体，他们不仅法律上高人一等，而且资格和才能上也高人一等。美丽强国网友先用所谓“平等”和“半斤对八两”把我的优势削平，再马上扔掉这些美丽词汇，把自己抬高到“觉悟的精英”这个高人一等的位置。 

这个某种潜意识是什么？就是不惜一切逻辑错误的代价，或牺牲历史事实本身，就是要非毛、反毛、丑化毛泽东和毛泽东时代。反毛精英知识分子不想、也不敢设立任何客观标准，只想凭自己信口雌黄，今天主观捏造一段历史，明天又主观捏造一段历史，并把捏造的历史，说成是“过来人的经历”，无可辩驳。要设立标准也行，那就把他们自己的个人（所谓“主体”）主观丑化设为客观的普遍的标准，并强加给社会，要别人接受。 

经过这样的分析，美丽强国网友包装在平等的美丽词汇中的精英论历史观，就露出或实实在在的狐狸尾巴。别人不接受精英论历史观怎么公办？ 当然就是愚昧，没有自我意识，就是工具。美丽强国网友的“自我意识”，“工具论”，就这样变得一文不值。我原来还以为他在自我意识的浅层意义上的认识，即“尊重个体的自我”方面，摸到了一点边，只是在深层次的“平等、自由、博爱”方面，浪费了亲身经历。现在看来，他老人家对自我意识/觉悟的浅层方面，也没有摸着边。其所谓自我意识，就是“我是目的，我是主体，我是标准，我就是一切，所以我说的就是对的”（主体就是标准，美丽强国是主体，所以美丽强国就是标准,就是对的）。 

这样我们就最终知道，近30年来人们通过主流媒体得到的“毛泽东时代”，是十足的反毛精英知识分子提供的“话语体系”，它是精英们把自己设定为主体，又把自己的主观认识设立为判断标准，恣意糟蹋歪曲毛泽东时代之后给出的“关于毛泽东时代话语体系”，与毛泽东时代自身的客观真实性没有关系。当然，作为对比，也为了衬托他们自己的标准是客观的，他们就对于毛泽东之前和之后的时代，同样以个人作为主体又作为标准，把一朵北京城里春天飘零的柳絮，说的也要比洛阳城的牡丹美丽万倍。 

既然吹出的美丽能够强国，毛泽东的斗争哲学，就必然贱如粪土无疑。美丽强国网友，对吗？  （且听下回分解）  右而左
