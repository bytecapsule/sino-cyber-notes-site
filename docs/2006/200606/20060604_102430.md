物先生的“不看广告看疗效”结论正确但分析错误 

我是同意物先生的不看广告看疗效的说法，那是“实践是检验真理的唯一标准”的另一种说法。但是看了物先生的陈词滥调之后，发现物先生在注意到“病人”疗效好的时候，却把“服的是哪种药”给搞错了。而且他的贴子具体都存在一些逻辑错误。 

物先生首先强调了一下各发达国家的福利，但是物先生就没有看过《红楼梦》吗？那大观园里的各个丫环小姐的福利也是相当不错的。物先生不认为中国现在的制度是民主制度，却认为现在全世界大多数国家都实行的民主制度，那么，是不是“民主”造成了福利，当然要统计所有的实行了“民主”的国家，而不仅仅是发达国家不是？你光说印度还不够啊，还有菲律宾啊，还有美国正在做的重要试验，伊拉克啊，还有伊朗，巴勒斯坦，等等，南斯拉夫不也是在“民主”制度下垮台的吗？ 

如果要看一味药的疗效，当然不能够专门选择治好了病的病人，而是要严肃地讨论所有服了这味药的病人，效果如何，这才能够得出结论。 

在《红楼梦》中刘佬佬参观了几回大观园，发现大观园里的人日子过得不错，比乡下人强多了，但是不应当误以为只要乡下人采用了大观园的制度，乡下人就都能够过得和贾宝玉一样的好。 

但是我在物先生的贴子中，注意到了这句话：“社会主义国家朝鲜和古巴的9年免费教育和免费医疗，我们却没有，这说明什么？说明这些社会福利的缺乏原因并不在于经济落后，而在于社会制度存在分配不公的问题。” 

这句话我倒是同意，因此朝鲜和古巴当然就是民主制度啦。物先生物必把这个结论坚持到底，以免得到时候又算糊涂帐。 

物先生是迷信选举的，那么，根据物先生的这个说法，当然朝鲜和古巴的领袖们也是选举产生的。 

而物先生说的“生活在俄罗斯的无国籍人和常驻俄罗斯的外国公民同俄罗斯公民一样,享有医疗保险制度中规定的权利”，则应当注意到俄罗斯的这种福利绝对不来源于俄罗斯的“民主”，而完全是前苏联遗留下来的制度。 

因此，鼓吹福利，这我非常赞成，我认为网上民主应当继续推动政府将中国打造成一个高福利的国家。 

从经济学上看，按我的观点，有可能福利越高，在一定程度上国家也会变得发达。现在国内的经济学家们，主流媒体，一提到欧洲的福利，就说德国如何如何焦头烂额啊，财政赤字啊。其实他们都忘了，德国再怎么焦头烂额也是发达国家，不可能因为福利制度的错误而导致德国变成了一个发展中国家。 

因此，我们从欧洲的福利制度中，应当回顾起，我们当初对计划经济的批判有可能错了，对社会主义大锅饭的批判，有可能错了，正如我前些日子的贴子的看法，一个已经使用了不可再生性能源的国家，在生产上是有很大的余量的，这个余量足够人们偷懒啊，上班随便啊，或者假期长长的啊，或者有人不上班光吃饭啊，经济往往不会因为这些情况而垮掉，反而有可能因为这些情况而发达起来。 

或者说，我对demmos先生的按人发钱方案是感兴趣的。 

其实，社会主义大锅饭没有错，制度是好的，只是要增加一点，允许私企合法化，那就成了。 

我国的制度当然是民主制度，但是，从我国的福利不行这一点看，我国的问题不在于民主制度，而在于福利制度，在福利制度上，我们的领导人可能被主流们误导了，当年大批大锅饭是没有必要的。 

何必让农民累得半死去搞联产承包？真想从农民的农业生产那里抠出来点什么吗？而现在明明已经连农业税都免了嘛，说明现在中央政府根本就不指望从农业生产那里得到什么了，还要补贴回去了。既然如此，农业生产搞社会主义大锅饭，大家没有积极性，本来就是无所谓的事情。就说南街村，不也是弄几个人站岗，说明南街村也人浮于事。现代化的生产是机器起作用，人应当是越来越悠闲的。 

中央政府在福利制度上的犹豫不决，完全是主流经济学家们误导的结果。是一种小农心理造成的结果。 

至于物先生硬要把什么问题的解决都归公于民主制度，那属于扯淡。比如说在苏联成立前的俄国，还有德国法西斯国家，日本法西斯国家，早早都实行了义务教育，那难道是人民抗争啊民主啊造成的么？根本就是扯淡嘛。 

也就是说，物先生注意到的病人疗效好，却没有发现真正使病人好转的药是什么，反正他满脑瓜子就是民主。这就象前些年伪科学盛行的时候抓到的那个胡万林胡大医，一帮人围着他说“灵啊，灵”，而这个胡大医的开药是非常简单的，就是所有的病人，不管他得的什么病，统统都是一种药，就是芒硝，物先生的欺骗和胡大医的办法差不多，但他用的“芒硝”就是“民主”。
