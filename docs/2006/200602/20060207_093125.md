介绍一下我是怎样在贴子中贴歌曲的   [数学]  2005-12-28 

有一些网友好象还不知道怎样在贴子中贴歌曲。当然，可以照强国论坛上指示的办法来贴歌曲，而我的关于超文本的知识并不是很够的，但也向这些不太知道的网友介绍一下，我是怎样贴歌曲。 

我的贴歌曲的办法和有一些网友的不同，那些网友的贴子，一打开就能够听得到乐曲。我觉得那样会给人吓一跳的感觉，甚至怀疑自己的计算机是不是被攻击了。因此，我通常是采用一个超级链接，让网友们单击一下，就能够听得到，如果不单击，当然就听不到，比如有的人就是不喜欢听，我就不好强让他听。所以我喜欢这样的办法。此外，我是懒人，懒人当然就不喜欢搞得太复杂，越简单越好。 

首先要问想贴歌曲的网友，你要贴的歌曲的文件，是不是在自己的CD碟上，或者在自己的计算机上存着？如果说是，那是不能够直接贴在贴子里的，因为，一般而言，我是要假设你的计算机不是一个网站，别人访问不到你的计算机。 

你一定要在互联网上找到一个网站，让那个网站给你提供一块存放文件的空间，然后你把你喜欢的歌曲，存放在你的计算机中的歌曲，传送到那个空间。 

那么，既然那个网站要给你一块空间，通常你是要付钱的，比如说什么新浪啊，网易啊，等等等等，许多网站都提供这种服务。付的钱多少，是和你申请的空间大小有关系的。但是，也有的网站不需要付钱，是免费的，那你得去找。 

找到了那样的网站，获得了一块空间后，具体怎样上传文件到你的那一块空间，那个网站就有具体的说明，你也可以直接打电话去那个网站问，特别是在你已经掏了钱的情况下，你是顾客，你是上帝嘛。 

请注意，如果你要掏钱，你就要年复一年地掏钱，这样你的文件才能够保存。什么时候你不掏钱了，你的那块空间就有可能消失了。如果你使用的是免费的空间，那就有一种危险，就是不知道什么时候网站不高兴让你免费使用了，你的文件也就消失了。 

你甚至死了以后，也还是要年复一年地委托它人帮你交钱，把那个网站提供的空间给维持下去。否则的话，比如说，你这一次贴了一个歌曲，大家听了觉得不错。结果五年以后，你的那个空间不在了，则有人在强国论坛检索五年前你的这个贴子的时候，就听不到歌曲了。 

强国论坛为什么不提供这样的空间？因为强国论坛的空间狭小，通常这种音乐或者图像甚至动画的空间，存储量都是很大的。因此，只能够通过强国论坛的贴子中的超级链接转到那个空间去。这样一来，你在强国论坛看到一个贴子里面有照片，其实，照片的图案并不存放在强国论坛，而是在另一个地方。 

但是，这里面有一个危险性存在，或者说，有一种黑客的手法存在，我在这里是写给强国论坛的版主们听的。希望版主们能够知道这个漏洞。当然，也许他们已经知道，只不过不管就是了，或者有其它的办法也说不定。 

比如说，一个人上了一个贴，贴子中有一个超级链接链往他的另一个网站上的空间中的一个图像文件，这样，大家就能够看得到一张照片了，可能这张照版经版主审查认为可以，就通过了。 

但是一年过后，或者几年过后，诸网友们可能要通过检索手段检索到这个贴子再拿来看一看的。如果这个网友故意捣鬼，在一年以后，把他的那个空间的图像文件给改了，名字没有改，但是内容却换了，比如说，成了一张反动口号的文字构成的照片，或者一张反动的宣传画。那么，版主们知道不知道呢？除非版主们对有超级链接的所有贴子反复地定期地进行审查，否则的话，沉到底下的贴子版主们不管，就有可能出现这种事情。 

好了，还是来对网友们说怎样搞歌曲的超级链接。 

现在假设你已经有了一个放在网站上的歌曲，或者，干脆就是某个音乐网站上的歌曲。这样，你就一定有一个这个文件的超级链接了。比方说，我这里有一个例子，我下面有一个我放在我朋友网站的一个文件，超级链接的网址是：（我用引号括起来）“http://math.shekou.com/back/我们共产党人.mid”，你把双引号里的内容直接在浏览器的地址栏上键入后再按回车键（Enter键），就能够听得到我编的这个MIDI音乐了，你在贴歌曲前一定要这样试一试，保证这个网址一定有效，这才做下一步。 

下一步就是你在你的贴子中，键入以下的一行字，这就构成了一个超级链接了，把这样的贴子上传到强国论坛，别人只要单击这个超级链接，就能够听到歌曲了。 

<A HREF="http://math.shekou.com/back/我们共产党人.mid">我们共产党人</A> 

请注意上面一句的两个双引号一定是直的，就是说不是汉字的双引号，最好你在英文输入状态下键入，尖括号也必须是英文状态下键入的，其它符号当然也一样，除了文件名中有中文只好在中文输入下键入。后面的那个“我们共产党人”你可以改成任何一段其它的什么字，是显示在文件中用的。 

当然，我的这个贴子中没有超级链接，你在上面乱点也没有用。不过如果你把我的这个贴子上面那一行样例拷贝到你的贴子中时，它有可能就变成超级链接了。
